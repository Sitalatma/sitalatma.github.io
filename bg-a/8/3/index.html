โศลก 3

śrī-bhagavān uvāca
 ศฺรี-ภควานฺ อุวาจ
 ศรี-ภะคะวาน อุวาจะ
 ชรี-บฺะกะวาน อุวาชะ
 akṣaraṁ brahma paramaṁ
 อกฺษรํ พฺรหฺม ปรมํ

 svabhāvo ’dhyātmam ucyate
 สฺวภาโว ’ธฺยาตฺมมฺ อุจฺยเต

 bhūta-bhāvodbhava-karo

śrī-bhagavān uvāca ศฺรี-ภควานฺ อุวาจ ศรี-ภะคะวาน อุวาจะ ชรี-บฺะกะวาน อุวาชะ — องค์ภควานตรัส, akṣaram อกฺษรมฺ อักษะรัม อัคชะรัม — ไม่มีวันถูกทำลาย, brahma พฺรหฺม พรัหมะ บระฮมะ — brahman พฺรหฺมนฺ พรัหมัน บระฮมัน, paramam ปรมมฺ ปะระมัม พะระมัม — ทิพย์, svabhāvaḥ สฺวภาวห์ สวะภาวะห์ สวะบฺาวะฮ — ธรรมชาติอมตะ, adhyātmam อธฺยาตฺมมฺ อัธยาตมัม อัดฺยาทมัม — ตัวชีวิต, ucyate อุจฺยเต อุจยะเต อุชยะเท — เรียกว่า, bhūta-bhāva-udbhava-karaḥ ภูต-ภาว-อุทฺภว-กรห์ ภูตะ-ภาวะ-อุทภะวะ-กะระห์ บํูทะ-บฺาวะ-อุดบฺะวะ-คะระฮ — ผลิตร่างวัตถุของสิ่งมีชีวิต, visargaḥ วิสรฺคห์ วิสัรคะห์ วิสารกะฮ — การ สร้าง, karma กรฺม กัรมะ คารมะ — กิจกรรมเพื่อผลทางวัตถุ, saṁjñitaḥ สํชฺญิตห์ สัญชญิตะห์ สัมกยิทะฮ — เรียกว่า

คำแปล

องค์ภควานตรัสว่า  ดวงชีวิตทิพย์ที่ไม่มีวันถูกทำลายเรียกว่า  brahman พฺรหฺมนฺ พรัหมัน บระฮมัน  และธรรมชาตินิรันดรของเขาเรียกว่า  adhyātma อธฺยาตฺม อัธยาตมะ อัดฺยาทมะ  หรือดวงชีวิต  กิจกรรมเกี่ยวกับการพัฒนาร่างกายวัตถุของสิ่งมีชีวิตเรียกว่ากรรม  (  karma กรฺม กัรมะ คารมะ  )  หรือกิจกรรมเพื่อผลทางวัตถุ

คำอธิบาย

brahman พฺรหฺมนฺ พรัหมัน บระฮมัน  ไม่มีวันถูกทำลายและเป็นอยู่ชั่วกัลปวสาน  สถานภาพเดิมแท้ของเขาไม่มีการเปลี่ยนแปลงไม่ว่าในขณะใด  แต่เหนือไปจาก  brahman พฺรหฺมนฺ พรัหมัน บระฮมัน  ยังมี  para-brahman,  brahman ปร-พฺรหฺมนฺ พฺรหฺมนฺ ปะระ-พรัหมัน พรัหมัน พะระ-บระฮมันบระฮมัน  หมายถึงสิ่งมีชีวิตและ  para-brahman ปร-พฺรหฺมนฺ ปะระ-พรัหมัน พะระ-บระฮมัน  หมายถึงองค์  bhagavān ภควานฺ ภะคะวาน บฺะกะวาน  สถานภาพพื้นฐานของสิ่งมีชีวิตแตกต่างไปจากสถานภาพที่เขาได้รับในโลกวัตถุ  ในวัตถุจิตสำนึกธรรมชาติคือพยายามเป็นเจ้าแห่งวัตถุ  แต่ในจิตสำนึกทิพย์หรือ  kṛṣṇa กฺฤษฺณ กฤษณะ คริชณะ  จิตสำนึกสถานภาพของเขาคือเป็นผู้รับใช้องค์  bhagavān ภควานฺ ภะคะวาน บฺะกะวาน  เมื่อสิ่งมีชีวิตอยู่ในวัตถุจิตสำนึก  จะต้องได้รับร่างกายต่าง  ๆ  ในโลกวัตถุ  เช่นนี้เรียกว่ากรรม  (  karma กรฺม กัรมะ คารมะ  )  หรือการสร้างอันหลากหลายด้วยการบังคับของจิตสำนึกวัตถุ

ในวรรณกรรมพระ  veda เวท เวทะ เวท  สิ่งมีชีวิตเรียกว่า  jīvātmā ชีวาตฺมา ชีวาตมา   และ  brahman พฺรหฺมนฺ พรัหมัน บระฮมัน  แต่ไม่เคยถูกเรียกว่า  para-brahman ปร-พฺรหฺมนฺ ปะระ-พรัหมัน พะระ-บระฮมัน  สิ่งมีชีวิต  (jīvātmā) ชีวาตฺมา ) ชีวาตมา ) )  ได้รับสถานภาพต่าง  ๆ  บางครั้งกลืนเข้าไปในธรรมชาติวัตถุอันมืดมนและสำคัญตนเองกับวัตถุ  และบางครั้งสำคัญตนเองกับสิ่งที่สูงกว่าคือธรรมชาติทิพย์  ดังนั้น  สิ่งมีชีวิตจึงได้ชื่อว่าเป็นพลังงานพรมแดนขององค์  bhagavān ภควานฺ ภะคะวาน บฺะกะวาน  ตามที่สำคัญตนเองกับธรรมชาติวัตถุหรือธรรมชาติทิพย์จึงได้รับร่างวัตถุหรือร่างทิพย์  ในธรรมชาติวัตถุเขาอาจได้รับร่างกายจากหนึ่งใน  8,400,000  เผ่าพันธุ์ชีวิตแต่ในธรรมชาติทิพย์จะมีเพียงร่างเดียว  ในธรรมชาติวัตถุบางครั้งปรากฏเป็นมนุษย์  เป็นเทวดา  เป็นสัตว์เดรัจฉาน  เป็นนก  ฯลฯ  ตามกรรมของตน  อาจบรรลุถึงดาวเคราะห์วัตถุบนสรวงสวรรค์  และได้รับความสุขกับสิ่งอำนวยความสะดวก  บางครั้งเขาปฏิบัติพิธีบูชา(yajña ยชฺญ ยัชญะ ยะกยะ  )  แต่เมื่อผลบุญหมดลงจะกลับมาในโลกนี้อีกครั้งในร่างมนุษย์  วิธีกรรมเช่นนี้เรียกว่ากรรมหรือ  (karma กรฺม กัรมะ คารมะ  )

chāndogya  upaniṣad ฉานฺโทคฺย อุปนิษทฺ ฉานโทคยะ อุปะนิษัท   อธิบายวิธีการทำพิธีบูชาทางพระ  veda เวท เวทะ เวท  ที่แท่นพิธีบูชามีการถวายสิ่งของห้าอย่างไปในไฟห้าชนิด  ไฟห้าชนิดสำเหนียกจาก  ดาวเคราะห์สวรรค์  เมฆโลก  ชาย  และหญิง  และสิ่งถวายในพิธีบูชาห้าอย่างคือ  ความศรัทธา  ผู้มีความสุขบนดวงจันทร์  ฝน  เมล็ดข้าว  และอสุจิ

ในพิธีการบูชา  สิ่งมีชีวิตทำพิธีบูชาโดยเฉพาะเพื่อบรรลุถึงสรวงสวรรค์  และก็บรรลุถึงจริงในเวลาต่อมา  เมื่อผลบุญแห่งการบูชาหมดสิ้นลง  สิ่งมีชีวิตตกลงมาบนโลกในรูปของฝน  จากนั้นมาอยู่ในรูปของเมล็ดข้าว  และมนุษย์รับประทานเมล็ดข้าวกลายเป็นอสุจิซึ่งทำให้ผู้หญิงตั้งครรภ์  ดังนั้น  สิ่งมีชีวิตได้รับร่างมนุษย์อีกครั้งหนึ่งเพื่อปฏิบัติพิธีบูชา  และวนเวียนอยู่ในวัฏจักรนี้  เช่นนี้สิ่งมีชีวิตจึงไป  ๆ  มา  ๆ  บนวิถีทางวัตถุชั่วกัลปวสาน  อย่างไรก็ดี  บุคคลใน  kṛṣṇa กฺฤษฺณ กฤษณะ คริชณะ  จิตสำนึกหลีกเลี่ยงการบูชาเช่นนี้  เขาปฏิบัติ  kṛṣṇa กฺฤษฺณ กฤษณะ คริชณะ  จิตสำนึกโดยตรง  ดังนั้น  จึงเตรียมตัวกลับคืนสู่เหย้าคืนสู่องค์  bhagavān ภควานฺ ภะคะวาน บฺะกะวาน  

นักวิจารณ์  bhagavad-gītā ภควทฺ-คีตา ภะคะวัท-คีตา ดฺยานะ-โยกะ  ผู้ไม่เชื่อในรูปลักษณ์คิดว่า  brahman พฺรหฺมนฺ พรัหมัน บระฮมัน  รับเอารูปของ  jīva ชีว ชีวะ จีวะ  ในโลกวัตถุอย่างไร้เหตุผล  และเพื่อสนับสนุนประเด็นนี้พวกเขาได้อ้างอิงบทที่สิบห้าโศลก  7  ของ  แต่ในโศลกนี้องค์  bhagavān ภควานฺ ภะคะวาน บฺะกะวาน  ยังตรัสอีกว่าสิ่งมีชีวิตเป็น  “ละอองอณูนิรันดรของตัวข้า”  สิ่งมีชีวิตเป็นละอองอณูขององค์  bhagavān ภควานฺ ภะคะวาน บฺะกะวาน  และอาจตกลงมาในโลกวัตถุ  แต่องค์bhagavān ภควานฺ ภะคะวาน บฺะกะวาน  (acyuta)( อจฺยุต ) ( อัจยุตะ ) (อัชยุทะ)  ไม่มีวันตกลงต่ำ  ดังนั้น  ข้อสรุปที่ว่า  brahman พฺรหฺมนฺ พรัหมัน บระฮมัน  สูงสุดรับเอารูปของ  jīva ชีว ชีวะ จีวะ  จึงยอมรับไม่ได้  มีความสำคัญที่ควรจำไว้ว่าในวรรณกรรมพระ  veda เวท เวทะ เวท  brahman พฺรหฺมนฺ พรัหมัน บระฮมัน  (สิ่งมีชีวิต)แตกต่างจาก  para-brahman ปร-พฺรหฺมนฺ ปะระ-พรัหมัน พะระ-บระฮมัน  (องค์  bhagavān ภควานฺ ภะคะวาน บฺะกะวาน  )
