ye เย เย เย — พวกเขาเหล่านี้, api อปิ อะปิ อพิ — เช่นกัน, anya อนฺย อันยะ อันยะ — ของผู้อื่น, devatā เทวตา เทวะตา เดวะทา — เหล่าเทวดา, bhaktāḥ ภกฺตาห์ ภักตาห์ บัฺคธาฮ — เหล่า สาวก, yajante ยชนฺเต ยะชันเต ยะจันเท — บูชา,śraddhayā anvitāḥ ศฺรทฺธยา อนฺวิตาห์ ศรัทธะยา อันวิตาห์ รัดดฺะยา อันวิทาฮ — ด้วยความศรัทธา, te เต เต เท — พวกเขา, api อปิ อะปิ อพิ — เช่น กัน, mām มามฺ มาม มาม — ข้า, eva เอว เอวะ เอวะ — เท่านั้น, kaunteya เกานฺเตย เกานเตยะ คะอุนเทยะ — โอ้ โอรสพระนาง kuntī กุนฺตี กุนตี คุนที, yajanti ยชนฺติ ยะชันติ ยะจันทิ — พวกเขาบูชา, avidhi-pūrvakam อวิธิ-ปูรฺวกมฺ อะวิธิ-ปูรวะกัม อวิดิฺ-พูรวะคัม — ในวิธีที่ผิด

คำแปล

พวกที่เป็นสาวกของเทวดาองค์อื่น ๆ และบูชาเทวดาด้วยความศรัทธา อันที่จริงบูชาข้าเท่านั้น โอ้ โอรสพระนาง kuntī กุนฺตี กุนตี คุนที แต่ทำไปในวิธีที่ผิด

คำอธิบาย

kṛṣṇa กฺฤษฺณ กฤษณะ คริชณะ ตรัสว่า “บุคคลผู้ปฏิบัติการบูชาเทวดาไม่มีปัญญาเท่าใดนัก ถึงแม้ว่าการบูชาเช่นนี้เป็นการถวายให้ข้าทางอ้อม” ตัวอย่างเช่น เมื่อมนุษย์รดน้ำไปที่ใบไม้และกิ่งก้านสาขาของต้นไม้โดยไม่รดน้ำไปที่ราก เขาทำไปโดยมีความรู้ไม่เพียงพอหรือไม่ปฏิบัติตามหลักการที่กำหนดไว้ ในทำนองเดียวกัน วิธีการรับใช้ส่วนต่าง ๆ ของร่างกายคือส่งอาหารไปที่ท้อง อันที่จริง เทวดาเป็นเจ้าพนักงานหรือเจ้าหน้าที่ต่าง ๆ ในรัฐบาลขององค์ bhagavān ภควานฺ ภะคะวาน บฺะกะวาน เราต้องปฏิบัติตามกฎหมายที่รัฐบาลเป็นผู้ออก ไม่ใช่ปฏิบัติตามกฎหมายที่เจ้าพนักงานหรือเจ้าหน้าที่เป็นผู้ออก ทำนองเดียวกัน ผู้ใดที่ถวายการบูชาต่อองค์ bhagavān ภควานฺ ภะคะวาน บฺะกะวาน เพียงผู้เดียว จะทำให้เจ้าพนักงานและเจ้าหน้าที่ต่าง ๆ ของพระองค์พึงพอใจโดยปริยาย เจ้าพนักงานและเจ้าหน้าที่ปฏิบัติตนในฐานะที่เป็นผู้แทนของรัฐบาล การให้สินบนกับเจ้าหน้าที่และเจ้าพนักงานนั้นผิดกฎหมาย ได้กล่าวไว้ ณ ที่นี้ว่า avidhi-pūrvakam อวิธิ-ปูรฺวกมฺ อะวิธิ-ปูรวะกัม อวิดิฺ-พูรวะคัม อีกนัยหนึ่ง kṛṣṇa กฺฤษฺณ กฤษณะ คริชณะ ทรงไม่เห็นด้วยกับการบูชาเทวดาโดยไม่จำเป็น
