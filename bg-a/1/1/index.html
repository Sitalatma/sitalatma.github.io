dhṛtarāṣṭraḥ uvāca ธฺฤตราษฺฏฺรห์ อุวาจ ธฤตะราษฏระห์ อุวาจะ ดฺริทะราชทระฮ อุวาชะ — กษัตริย์ dhṛtarāṣṭra ธฺฤตราษฺฏฺร ธฤตะราษฏระ ดฺริทะรชทระ ตรัส, dharma-kṣetre ธรฺม-กฺเษเตฺร ธัรมะ-กเษตเร ดฺารมะ-คเชเทร — ในสถานที่ศักดิ์สิทธิ์, kuru-kṣetre กุรุ-กฺเษเตฺร กุรุ-กเษตเร คุรุ-คเชเทร — ชื่อ kurukṣetra กุรุกฺเษตฺร กุรุกเษตระ คุรุคเชทระ, samavetāḥ สมเวตาห์ สะมะเวตาห์ สะมะเวทาฮ — มาชุมนุมกัน, yuyutsavaḥ ยุยุตฺสวห์ ยุยุตสะวะห์ ยุยุทสะวะฮ — ปรารถนาจะสู้รบ, māmakāḥ มามกาห์ มามะกาห์ มามะคาฮ — ฝ่ายของข้า (เหล่าโอรส), pāṇḍavāḥ ปาณฺฑวาห์ ปาณฑะวาห์ พาณดะวาฮ — โอรสของ pāṇḍu ปาณฺฑุ ปาณฑุ พณดุ, ca จ จะ ชะ — และ, eva เอว เอวะ เอวะ — แน่นอน, kim กิมฺ กิม คิม — อะไร, akurvata อกุรฺวต อะกุรวะตะ อคุรวะทะ — พวกเขาได้ทำ, sañjaya สญฺชย สัญชะยะ สันจะยะ — โอ้ สันจะยะ

คำแปล

dhṛtarāṣṭra ธฺฤตราษฺฏฺร ธฤตะราษฏระ ดฺริทะรชทระ ตรัสว่า “โอ้ sañjaya สญฺชย สัญชะยะ สันจะยะ หลังจากบรรดาโอรสของข้าและโอรสของpāṇḍu ปาณฺฑุ ปาณฑุ พณดุ มาชุมนุมกันยังสถานที่ศักดิ์สิทธิ์แห่ง kurukṣetra กุรุกฺเษตฺร กุรุกเษตระ คุรุคเชทระ มีความปรารถนาจะสู้รบพวกเขาทำอะไรกัน?”

คำอธิบาย

bhagavad-gītā ภควทฺ-คีตา ภะคะวัท-คีตา ดฺยานะ-โยกะ ศาสตร์แห่งองค์ bhagavān ภควานฺ ภะคะวาน บฺะกะวาน ที่สรุปใน gītā-māhātmya คีตา-มาหาตฺมฺย คีตา-มาหาตมยะ คีตา - มาฮาทมยะ (คำสรรเสริญ bhagavad-gītā ภควทฺ-คีตา ภะคะวัท-คีตา ดฺยานะ-โยกะ ) ที่ได้อ่านกันอย่างแพร่หลาย กล่าวไว้ว่า เราควรอ่าน bhagavad-gītā ภควทฺ-คีตา ภะคะวัท-คีตา ดฺยานะ-โยกะ อย่างละเอียดถี่ถ้วนพร้อมกับการช่วยเหลือจากสาวกของ śrī ศฺรี ศรี ชรี kṛṣṇa กฺฤษฺณ กฤษณะ คริชณะ และพยายามเข้าใจโดยปราศจากการตีความจากแรงกระตุ้นส่วนตัว ตัวอย่างการเข้าใจอย่างชัดเจนอยู่ใน bhagavad-gītā ภควทฺ-คีตา ภะคะวัท-คีตา ดฺยานะ-โยกะ วิธีที่ arjuna อรฺชุน อัรชุนะ อารจุนะ ทรงเข้าใจ ซึ่งได้ยินมาจากองค์ bhagavān ภควานฺ ภะคะวาน บฺะกะวาน โดยตรง หากผู้ใดโชคดีพอที่มาเข้าใจ bhagavad-gītā ภควทฺ-คีตา ภะคะวัท-คีตา ดฺยานะ-โยกะ ตามสาย paramparā ปรมฺปรา ปะรัมปะรา มพะรา โดยไม่มีแรงกระตุ้นในการตีความจะบรรลุถึงการศึกษาปรัชญาพระ veda เวท เวทะ เวท ทั้งหมด และพระคัมภีร์ทั้งหมดในโลก ใน bhagavad-gītā ภควทฺ-คีตา ภะคะวัท-คีตา ดฺยานะ-โยกะ เราจะพบทุกสิ่งทุกอย่างที่มีอยู่ในพระคัมภีร์เล่มอื่น ๆ ผู้อ่านจะพบสิ่งที่หาจากที่อื่นไม่ได้ นี่คือมาตรฐานโดยเฉพาะของ bhagavad-gītā, bhagavad-gītā ภควทฺ-คีตา ภควทฺ-คีตา ภะคะวัท-คีตา ภะคะวัท-คีตา บฺะกะวัด - คีตา บฺะกะวัด - คีตา เป็นศาสตร์แห่งองค์ bhagavān ภควานฺ ภะคะวาน บฺะกะวาน ที่สมบูรณ์ซึ่งบุคลิกภาพสูงสุดแห่งพระเจ้า śrī ศฺรี ศรี ชรี kṛṣṇa กฺฤษฺณ กฤษณะ คริชณะ ทรงเป็นผู้ตรัสโดยตรง

ประเด็นที่สนทนากันระหว่าง dhṛtarāṣṭra ธฺฤตราษฺฏฺร ธฤตะราษฏระ ดฺริทะรชทระ และ sañjaya สญฺชย สัญชะยะ สันจะยะ ดังที่อธิบายไว้ใน mahābhārata มหาภารต มะหาภาระตะ มหาภารตะ เป็นหลักพื้นฐานแห่งปรัชญาอันยิ่งใหญ่นี้ เป็นที่เข้าใจว่าปรัชญานี้เกิดขึ้นที่สมรภูมิ kurukṣetra กุรุกฺเษตฺร กุรุกเษตระ คุรุคเชทระ ซึ่งเป็นสถานที่ศักดิ์สิทธิ์ของนักบุญ ตั้งแต่สมัยยุคพระ veda เวท เวทะ เวท โบราณ องค์ bhagavān ภควานฺ ภะคะวาน บฺะกะวาน ตรัสในขณะที่เสด็จลงมาบนโลกนี้ด้วยพระองค์เองเพื่อนำทางมนุษยชาติ

คำว่า dharma-kṣetra ธรฺม-กฺเษตฺร ธัรมะ-กเษตระ ดฺารมะ - คเชทระ (สถานที่สำหรับทำพิธีกรรมทางศาสนา) มีความสำคัญเพราะว่าที่สมรภูมิ kurukṣetra กุรุกฺเษตฺร กุรุกเษตระ คุรุคเชทระ บุคลิกภาพสูงสุดแห่งพระเจ้าทรงปรากฏอยู่ฝ่ายของ arjuna อรฺชุน อัรชุนะ อารจุนะ dhṛtarāṣṭra ธฺฤตราษฺฏฺร ธฤตะราษฏระ ดฺริทะรชทระ พระบิดาแห่งราชวงศ์ kuru กุรุ กุรุ คุรุ ทรงสงสัยเป็นอย่างยิ่งถึงชัยชนะขั้นสุดท้ายของโอรสของพระองค์ ด้วยความสงสัยนี้จึงตรัสถามเลขา sañjaya สญฺชย สัญชะยะ สันจะยะ ว่า “พวกเขาทำอะไรกัน?” พระองค์ทรงมั่นใจว่า โอรสของพระองค์และโอรสของพระอนุชา pāṇḍu ปาณฺฑุ ปาณฑุ พณดุ ได้มาชุมนุมกันที่สมรภูมิ kurukṣetra กุรุกฺเษตฺร กุรุกเษตระ คุรุคเชทระ ด้วยความมั่นใจในการทำศึกสงค rāma ราม รามะ ราม แต่คำถามนี้สำคัญเพราะพระองค์ทรงไม่ปรารถนาให้ญาติพี่น้องทั้งสองฝ่ายประนีประนอมกันทรงประสงค์จะทราบชะตากรรมของเหล่าโอรสอย่างชัดเจนในสนามรบ สงค rāma ราม รามะ ราม ได้เตรียมให้เกิดขึ้นที่สมรภูมิ kurukṣetra กุรุกฺเษตฺร กุรุกเษตระ คุรุคเชทระ คัมภีร์พระ veda เวท เวทะ เวท ได้กล่าวว่าเป็นสถานที่สักการะบูชาของเหล่าเทวดาบนสรวงสวรรค์ด้วย dhṛtarāṣṭra ธฺฤตราษฺฏฺร ธฤตะราษฏระ ดฺริทะรชทระ ทรงรู้สึกกลัวมากเกี่ยวกับอิทธิพลของสถานที่ศักดิ์สิทธิ์ที่จะส่งผลในสมรภูมิ ทรงทราบดีว่าสิ่งนี้จะมีอิทธิพลส่งเสริมให้arjuna อรฺชุน อัรชุนะ อารจุนะ และบรรดาบุตรของ pāṇḍu ปาณฺฑุ ปาณฑุ พณดุ ไปในทางที่ดี เนื่องจากทุกพระองค์ในราชวงค์ pāṇḍu ปาณฺฑุ ปาณฑุ พณดุทรงมีคุณธรรมโดยธรรมชาติ sañjaya สญฺชย สัญชะยะ สันจะยะ เป็นศิษย์ของ vyāsa วฺยาส วยาสะ วิยาสะ ดังนั้น ด้วยพระเมตตาของ vyāsa วฺยาส วยาสะ วิยาสะ sañjaya สญฺชย สัญชะยะ สันจะยะ จึงสามารถเห็นสมรภูมิ kurukṣetra กุรุกฺเษตฺร กุรุกเษตระ คุรุคเชทระ แม้ขณะอยู่ในห้องของ dhṛtarāṣṭra ธฺฤตราษฺฏฺร ธฤตะราษฏระ ดฺริทะรชทระ ดังนั้น dhṛtarāṣṭra ธฺฤตราษฺฏฺร ธฤตะราษฏระ ดฺริทะรชทระ จึงทรงถาม sañjaya สญฺชย สัญชะยะ สันจะยะ เกี่ยวกับสถานการณ์ที่สนามรบ

ทั้งบรรดา pāṇḍava ปาณฺฑว ปาณฑะวะ พาณดะวะ และบรรดาโอรสของ dhṛtarāṣṭra ธฺฤตราษฺฏฺร ธฤตะราษฏระ ดฺริทะรชทระ ทรงอยู่ในราชวงศ์เดียวกัน แต่จิตใจของ dhṛtarāṣṭra ธฺฤตราษฺฏฺร ธฤตะราษฏระ ดฺริทะรชทระ ทรงถูกเปิดเผย ณ ที่นี้ ว่าทรงตั้งใจที่จะอ้างสิทธิ์เหล่าโอรสของพระองค์ว่าเป็น kuru กุรุ กุรุ คุรุ เท่านั้น และตัดพวกโอรสของ pāṇḍu ปาณฺฑุ ปาณฑุ พณดุ ให้ออกจากกองมรดกแห่งราชวงศ์ เช่นนี้ เราจึงเข้าใจสถานภาพอันแท้จริงของ dhṛtarāṣṭra ธฺฤตราษฺฏฺร ธฤตะราษฏระ ดฺริทะรชทระ ในความสัมพันธ์กับโอรสของ pāṇḍu ปาณฺฑุ ปาณฑุ พณดุ ซึ่งเป็นหลานของพระองค์ เสมือนหนึ่งในทุ่งนาที่วัชพืชต้องถูกกำจัดฉะนั้น เป็นที่คาดหวังตั้งแต่ตอนเริ่มต้นแล้วว่า ณ ศาสนสถานแห่ง kurukṣetra กุรุกฺเษตฺร กุรุกเษตระ คุรุคเชทระ ที่พระบิดาแห่งศาสนา śrī ศฺรี ศรี ชรี kṛṣṇa กฺฤษฺณ กฤษณะ คริชณะ ทรงปรากฏ duryodhana ทุโรฺยธน ทุรโยธะนะ ดุรโยดฺะนะ และพรรคพวกเปรียบเสมือนวัชพืชที่จะต้องถูกกำจัดทั้งหมด และผู้ทรงธรรมซึ่งนำโดย yudhiṣṭhira ยุธิษฺฐิร ยุธิษฐิระ ยุดิฺชทิฺระ จะได้รับการสถาปนาโดยองค์ bhagavān ภควานฺ ภะคะวาน บฺะกะวาน นี่คือ ความสำคัญของคำว่า dharma-kṣetre ธรฺม-กฺเษเตฺร ธัรมะ-กเษตเร ดฺารมะ-คเชเทร และ kuru-kṣetre กุรุ-กฺเษเตฺร กุรุ-กเษตเร คุรุ-คเชเทร นอกจากความสำคัญทางประวัติศาสตร์และทางคัมภีร์พระ veda เวท เวทะ เวท
