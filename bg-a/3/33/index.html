ภควัต-คีตา ฉบับเดิม

บทที่ สาม

กรฺม-โยค

โศลก 33

sadṛśaṁ ceṣṭate svasyāḥ
 สทฺฤศํ เจษฺฏเต สฺวสฺยาห์
 สัทฤศัง เจษฏะเต สวัสยาห์
 สะดริชัม เชชทะเท สวัสยาฮ
 prakṛter jñānavān api
 ปฺรกฺฤเตรฺ ชฺญานวานฺ อปิ
 ปรักฤเตร ชญานะวาน อะปิ
 พระคริเทร กยานะวาน อพิฺ
 prakṛtiṁ yānti bhūtāni
 ปฺรกฺฤตึ ยานฺติ ภูตานิ
 ปรักฤติง ยานติ ภูตานิ
 พระคริทิม ยานทิ บํูทานิ
 nigrahaḥ kiṁ kariṣyati
 นิคฺรหห์ กึ กริษฺยติ
 นิคระหะห์ กิง กะริษยะติ
 นิกระฮะฮ คิม คาริชยะทิฺ

sadṛśam สทฺฤศมฺ สัทฤศัม สะดริชัม — ตามนั้น, ceṣṭate เจษฺฏเต เจษฏะเต เชชทะเท — พยายาม, svasyāḥ สฺวสฺยาห์ สวัสยาห์ สวัสยาฮ — ด้วยตัวเขาเอง, prakṛteḥ ปฺรกฺฤเตห์ ปรักฤเตห์ พระคริเทฮ — ระดับ ของธรรมชาติ, jñāna-vān ชฺญาน-วานฺ ชญานะ-วาน กยานะ-วาน — มีความรู้, api อปิ อะปิ อพิ — ถึงแม้ว่า, prakṛtim ปฺรกฺฤติมฺ ปรักฤติม พระคริทิม — ธรรมชาติ, yānti ยานฺติ ยานติ ยานทิ — ได้ รับ,bhūtāni ภูตานิ ภูตานิ บํูทานิ — มวลสิ่งมีชีวิต, nigrahaḥ นิคฺรหห์ นิคระหะห์ นิกระฮะฮ — ความอดกลั้น, kim กิมฺ กิม คิม — อะไร, kariṣyati กริษฺยติ กะริษยะติ คาริชยะทิ — สามารถทำ

คำแปล

แม้ผู้รู้ยังต้องปฏิบัติตามธรรมชาติของตนเอง เพราะทุกคนปฏิบัติตามธรรมชาติที่ตนได้รับมาจากสามระดับ การเก็บกดเอาไว้จะได้รับผลสำเร็จอันใด?

คำอธิบาย

นอกเสียจากว่าเราจะสถิตในระดับทิพย์แห่ง kṛṣṇa กฺฤษฺณ กฤษณะ คริชณะ จิตสำนึก มิฉะนั้น เราจะไม่สามารถมีอิสรภาพจากอิทธิพลของระดับแห่งธรรมชาติวัตถุ ดังที่ องค์ bhagavān ภควานฺ ภะคะวาน บฺะกะวาน ได้ทรงยืนยันไว้ในบทที่เจ็ด (7.14) ฉะนั้น แม้ผู้มีการศึกษาสูงสุดทางโลกก็ยังหลุดพ้นจากบ่วงของ māyā มายา มายา มายา ไม่ได้ ด้วยความรู้ทางทฤษฎีหรือด้วยการแยกดวงวิญญาณออกจากร่างกายมีผู้ที่สมมุติว่าเป็นนักทิพย์นิยมมากมาย ภายนอกวางตัวว่ามีความเจริญในศาสตร์นี้แต่ภายในหรือส่วนตัวยังอยู่ภายใต้ระดับใดระดับหนึ่งของธรรมชาติวัตถุอย่างราบคาบ ซึ่งตนเองไม่สามารถข้ามพ้นไปได้ ในเชิงวิชาการเขาอาจจะเป็นผู้มีการศึกษาสูงมาก แต่เนื่องจากมาคลุกคลีอยู่กับธรรมชาติวัตถุอันแสนจะยาวนานจึงถูกพันธนาการ kṛṣṇa กฺฤษฺณ กฤษณะ คริชณะจิตสำนึกสามารถช่วยให้เราออกจากพันธนาการทางวัตถุได้ ถึงแม้ว่าเราจะปฏิบัติตามหน้าที่ที่กำหนดไว้ตามความเป็นอยู่ทางวัตถุ ฉะนั้น หากไม่มี kṛṣṇa กฺฤษฺณ กฤษณะ คริชณะ จิตสำนึกอย่างสมบูรณ์ เราไม่ควรยกเลิกอาชีพการงาน ไม่มีผู้ใดควรยกเลิกหน้าที่ที่ได้กำหนดไว้โดยฉับพลัน และมาสมมุติตนเองว่าเป็น yoga โยค โยคะ โยกะ หรือนักทิพย์นิยมแบบผิดธรรมชาติ สถิตในสถานภาพของตนเองและพยายามบรรลุถึง kṛṣṇa กฺฤษฺณ กฤษณะ คริชณะ จิตสำนึกภายใต้การฝึกฝนที่สูงยังจะดีกว่า เช่นนี้ เราอาจได้รับอิสรภาพจากเงื้อมมือพระนาง māyā มายา มายา มายา ซึ่งก็เป็นผู้รับใช้ของkṛṣṇa กฺฤษฺณ กฤษณะ คริชณะ เช่นกัน
