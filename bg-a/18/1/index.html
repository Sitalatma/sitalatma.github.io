โศลก 1

arjuna uvāca

 tattvam icchāmi veditum
 ตตฺตฺวมฺ อิจฺฉามิ เวทิตุมฺ

arjunaḥ uvāca อรฺชุนห์ อุวาจ อัรชุนะห์ อุวาจะ อารจุนะฮ อุวาชะ — arjuna อรฺชุน อัรชุนะ อารจุนะ ตรัส, sannyāsasya สนฺนฺยาสสฺย สันนยาสัสยะ สันนยาสัสยะ — ของการเสียสละ, mahā-bāho มหา-พาโห มะหา-พาโห มะฮา-บาโฮ — โอ้ นักรบผู้ยอดเยี่ยม, tattvam ตตฺตฺวมฺ ตัตตวัม ทัททวัม — ความจริง, icchāmi อิจฺฉามิ อิจฉามิ อิชชฺามิ — ข้าปรารถนา, veditum เวทิตุมฺ เวทิตุม เวดิทุม — เข้าใจ, tyāgasya ตฺยาคสฺย ตยาคัสยะ ทยากัสยะ — การเสีย สละ, ca จ จะ ชะ — เช่นกัน, hṛṣīkeśa หฺฤษีเกศ หฤษีเกศะ ฮริชีเคชะ — โอ้ เจ้าแห่งประสาทสัมผัส, pṛthak ปฺฤถกฺ ปฤถัก พริทัฺค — แตกต่าง, keśi-niṣūdana เกศิ-นิษูทน เกศิ-นิษูทะนะ เคชิ-นิชูดะนะ — โอ้ ผู้สังหารมาร keśī เกศี เกศี เคชี

คำแปล

arjuna อรฺชุน อัรชุนะ อารจุนะ ตรัสว่า โอ้ ยอดนักรบ ข้าปรารถนาที่จะเข้าใจจุดมุ่งหมายแห่งการเสียสละ ( tyāga ตฺยาค ตยาคะ ทยากะ ) และชีวิตสละโลก ( sannyāsa สนฺนฺยาส สันนยาสะ สันนยาสะ ) โอ้ ผู้สังหารมาร keśī เกศี เกศี เคชี โอ้ เจ้าแห่งประสาทสัมผัส

คำอธิบาย

อันที่จริง bhagavad-gītā ภควทฺ-คีตา ภะคะวัท-คีตา ดฺยานะ-โยกะ จบลงในบทที่สิบเจ็ด บทที่สิบแปดเป็นบทสรุปเสริมประเด็นต่าง ๆ ที่ได้กล่าวมาแล้ว ทุกบทของ bhagavad-gītā ภควทฺ-คีตา ภะคะวัท-คีตา ดฺยานะ-โยกะ องค์ śrī ศฺรี ศรี ชรี kṛṣṇa กฺฤษฺณ กฤษณะ คริชณะ ทรงเน้นการอุทิศตนเสียสละรับใช้ต่อบุคลิกภาพสูงสุดว่าเป็นจุดมุ่งหมายสูงสุดของชีวิต ประเด็นเดียวกันนี้ได้สรุปไว้ในบทที่สิบแปดว่าเป็นวิถีทางแห่งความรู้ที่ลับที่สุด หกบทแรกเน้นถึงการอุทิศตนเสียสละรับใช้ yoginām api sarveṣām… โยคินามฺ อปิ สเรฺวษามฺ … โยคินาม อะปิ สัรเวษาม … โยกินาม อพิ สารเวชาม… “ในบรรดา yoga โยค โยคะ โยกะ หรือนักทิพย์นิยมทั้งหลาย ผู้ที่ระลึกถึงข้าอยู่ภายในใจเสมอ ดีที่สุด” หกบทต่อมาได้กล่าวถึงการอุทิศตนเสียสละรับใช้ที่บริสุทธิ์พร้อมทั้งธรรมชาติและกิจกรรม และหกบทสุดท้ายอธิบายถึงความรู้ การเสียสละ กิจกรรมของธรรมชาติวัตถุและธรรมชาติทิพย์ และการอุทิศตนเสียสละรับใช้ สรุปได้ว่า การปฏิบัติทั้งหมดควรทำไปในความสัมพันธ์กับองค์ bhagavān ภควานฺ ภะคะวาน บฺะกะวาน ที่มีคำว่า oṁ tat sat โอํ ตตฺ สตฺ โอง ตัต สัต โอม ทัท สัท เป็นผู้แทน ซึ่งแสดงถึงพระ viṣṇu วิษฺณุ วิษณุ วิชณุ บุคลิกภาพสูงสุด ส่วนที่สามของ bhagavad-gītā ภควทฺ-คีตา ภะคะวัท-คีตา ดฺยานะ-โยกะ ได้แสดงว่า การอุทิศตนเสียสละรับใช้คือจุดมุ่งหมายสูงสุดของชีวิต มิใช่สิ่งอื่นใด ได้สถาปนาเช่นนี้โดยอ้างถึง ācārya อาจารฺย อาจารยะ อาชารยะ ในอดีต และ brahma-sūtra พฺรหฺม-สูตฺร พรัหมะ-สูตระ บระฮมะ-สูทระ, vedānta-sūtra เวทานฺต-สูตฺร เวทานตะ-สูตระ เวดานธะ - สูทระ พวกไม่เชื่อในรูปลักษณ์บางกลุ่มพิจารณาว่าพวกตนมีความรู้ใน vedānta-sūtra เวทานฺต-สูตฺร เวทานตะ-สูตระ เวดานธะ - สูทระ แต่เพียงผู้เดียว อันที่จริง vedānta-sūtra เวทานฺต-สูตฺร เวทานตะ-สูตระ เวดานธะ - สูทระ หมายไว้เพื่อให้เข้าใจการอุทิศตนเสียสละรับใช้เพราะว่าพระองค์ทรงเป็นผู้รวบรวมและทรงเป็นผู้รู้ vedānta-sūtra เวทานฺต-สูตฺร เวทานตะ-สูตระ เวดานธะ - สูทระ ซึ่งได้อธิบายไว้ในบทที่สิบห้า ในทุกคัมภีร์และในพระ veda เวท เวทะ เวท ทุกเล่ม การอุทิศตนเสียสละรับใช้คือวัตถุประสงค์ ประเด็นนี้ได้อธิบายไว้ใน bhagavad-gītā ภควทฺ-คีตา ภะคะวัท-คีตา ดฺยานะ-โยกะ 

ดังที่บทที่สองได้ประมวลเรื่องราวทั้งหมดที่กล่าวมา บทที่สิบแปดก็เช่นกันได้สรุปถึงคำสั่งสอนทั้งหมดที่ให้ไว้ แสดงให้เห็นถึงจุดมุ่งหมายของชีวิตคือการเสียสละและบรรลุถึงสถานภาพทิพย์เหนือสามระดับแห่งธรรมชาติวัตถุ arjuna อรฺชุน อัรชุนะ อารจุนะ ทรงปรารถนาที่จะทำให้ทั้งสองประเด็นของ bhagavad-gītā ภควทฺ-คีตา ภะคะวัท-คีตา ดฺยานะ-โยกะ คือการเสียสละ (tyāga)( ตฺยาค ) ( ตยาคะ ) ( ทยากะ ) และชีวิตสละโลก (sannyāsa)( สนฺนฺยาส ) ( สันนยาสะ ) ( สันนยาสะ ) กระจ่างขึ้น ดังนั้น ท่านจึงถามถึงความหมายของสองคำนี้

สองคำที่ใช้ในโศลกนี้แสดงถึงองค์ bhagavān ภควานฺ ภะคะวาน บฺะกะวาน คือคำ hṛṣīkeśa หฺฤษีเกศ หฤษีเกศะ ฮริชีเคชะ และ keśi-niṣūdana เกศิ-นิษูทน เกศิ-นิษูทะนะ เคชิ-นิชูดะนะ มีความสำคัญ hṛṣīkeśa หฺฤษีเกศ หฤษีเกศะ ฮริชีเคชะ คือ kṛṣṇa กฺฤษฺณ กฤษณะ คริชณะ เจ้าแห่งประสาทสัมผัสทั้งหมด ผู้ที่สามารถช่วยเราให้มีความสงบภายในใจเสมอ arjuna อรฺชุน อัรชุนะ อารจุนะ ทรงขอร้องให้พระองค์ทรงสรุปทุกสิ่งทุกอย่างที่จะทำให้เกิดความมั่นใจ เนื่องจาก arjuna อรฺชุน อัรชุนะ อารจุนะ ทรงมีข้อสงสัยบางประการและข้อสงสัยเปรียบเสมือนมาร ดังนั้น ท่านจึงเรียก kṛṣṇa กฺฤษฺณ กฤษณะ คริชณะ ว่า keśi-niṣūdana, keśi เกศิ-นิษูทน เกศิ เกศิ-นิษูทะนะ เกศิ เกศี-นิชูดะนะ เกศี เป็นมารที่น่ารังเกียจที่สุดซึ่งพระองค์ทรงสังหาร บัดนี้ arjuna อรฺชุน อัรชุนะ อารจุนะ ทรงคาดหวังให้ kṛṣṇa กฺฤษฺณ กฤษณะ คริชณะ สังหารมารแห่งความสงสัย
