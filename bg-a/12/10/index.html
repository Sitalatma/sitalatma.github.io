ภควัต-คีตา ฉบับเดิม

บทที่ สิบสอง

การอุทิศตนเสียสละรับใช้

โศลก 10

abhyāse ’py asamartho ’si
 อภฺยาเส ’ปฺยฺ อสมรฺโถ ’สิ
 อัภยาเส ’ปย อะสะมัรโถ ’สิ
 อับฺยาเส ่พิ อสะมารโทฺ ่สิ
 mat-karma-paramo bhava
 มตฺ-กรฺม-ปรโม ภว
 มัต-กัรมะ-ปะระโม ภะวะ
 มัท-คารมะ-พะระโม บฺะวะ
 mad-artham api karmāṇi
 มทฺ-อรฺถมฺ อปิ กรฺมาณิ
 มัท-อัรถัม อะปิ กัรมาณิ
 มัด-อารทัฺม อพิ คารมาณิ
 kurvan siddhim avāpsyasi
 กุรฺวนฺ สิทฺธิมฺ อวาปฺสฺยสิ
 กุรวัน สิทธิม อะวาปสยะสิ
 คุรวัน สิดดิฺม อวาพสยะสิ

abhyāse อภฺยาเส อัภยาเส อับฺยาเส — ในการปฏิบัติ, api อปิ อะปิ อพิ — ถึงแม้ว่า, asamarthaḥ อสมรฺถห์ อะสะมัรถะห์ อสะมารทฺะฮ — ไม่สามารถ, asi อสิ อะสิ อสิ — เธอเป็น, mat-karma มตฺ-กรฺม มัต-กัรมะ มัท-คารมะ — งานของข้า, paramaḥ ปรมห์ ปะระมะห์ พะระมะฮ — อุทิศตนเสียสละแด่, bhava ภว ภะวะ บฺะวะ — มาเป็น, mat-artham มตฺ-อรฺถมฺ มัต-อัรถัม มัท-อารทัฺม — เพื่อ ประโยชน์ของข้า, api อปิ อะปิ อพิ — แม้, karmāṇi กรฺมาณิ กัรมาณิ คารมาณิ — งาน, kurvan กุรฺวนฺ กุรวัน คุรวัน — ปฏิบัติ, siddhim สิทฺธิมฺ สิทธิม สิดดิฺม — สมบูรณ์, avāpsyasi อวาปฺสฺยสิ อะวาปสยะสิ อวาพสยะสิ — เธอจะบรรลุ

คำแปล

หากไม่สามารถปฏิบัติตามกฎเกณฑ์ของ bhakti ภกฺติ ภักติ บัฺคธิ - yoga โยค โยคะ โยกะ เธอก็พยายามทำงานให้แก่ข้า เพราะจากการทำงานให้ข้า เธอจะมาถึงระดับที่สมบูรณ์

คำอธิบาย

ผู้ที่ไม่สามารถปฏิบัติตามหลักธรรมของ bhakti-yoga ภกฺติ-โยค ภักติ-โยคะ บัฺคธิ-โยกะ ภายใต้การแนะนำของพระอาจารย์ทิพย์ ยังถูกนำพาให้มาถึงระดับสมบูรณ์นี้ได้ด้วยการทำงานให้แด่องค์ bhagavān ภควานฺ ภะคะวาน บฺะกะวาน จะทำงานนี้ได้อย่างไรได้อธิบายไว้แล้วในโศลกที่ห้าสิบห้าของบทที่สิบเอ็ด เราควรเห็นใจในการเผยแพร่ kṛṣṇa กฺฤษฺณ กฤษณะ คริชณะ จิตสำนึก มีสาวกมากมายปฏิบัติตนในการเผยแพร่kṛṣṇa กฺฤษฺณ กฤษณะ คริชณะ จิตสำนึกซึ่งต้องการความช่วยเหลือ ดังนั้น แม้เราไม่สามารถปฏิบัติตามหลักธรรมของ bhakti-yoga ภกฺติ-โยค ภักติ-โยคะ บัฺคธิ-โยกะ โดยตรง เราอาจพยายามช่วยงานนี้ได้ การริเริ่มกระทำสิ่งใดจำเป็นต้องใช้ที่ดิน เงินทุน องค์กร และแรงงาน เช่นเดียวกับธุรกิจ เราจำเป็นต้องมีสถานที่อยู่อาศัย มีเงินทุนสำหรับใช้จ่าย มีแรงงาน และมีองค์กรเพื่อขยาย สิ่งต่าง ๆ เหล่านี้มีความจำเป็นในการรับใช้ kṛṣṇa กฺฤษฺณ กฤษณะ คริชณะ ข้อแตกต่างก็คือ ในลัทธิวัตถุนิยมงานทำไปเพื่อสนองประสาทสัมผัส อย่างไรก็ดีงานที่คล้ายกันนี้สามารถกระทำได้เพื่อความพึงพอพระทัยของ kṛṣṇa กฺฤษฺณ กฤษณะ คริชณะ และนั่นคือกิจกรรมทิพย์ หากมีเงินเพียงพอเราสามารถช่วยก่อสร้างสำนักงาน หรือสร้างวัดเพื่อเผยแพร่ kṛṣṇa กฺฤษฺณ กฤษณะ คริชณะ จิตสำนึก หรืออาจช่วยในการพิมพ์หนังสือยังมีกิจกรรมอื่น ๆ อีกมากมายที่น่าสนใจ หากเราไม่สามารถสละผลของกิจกรรมเราก็สามารถสละบางส่วนเพื่อเผยแพร่ kṛṣṇa กฺฤษฺณ กฤษณะ คริชณะ จิตสำนึก การอาสาบ ṛg ฤคฺ ฤค ริก ารรับใช้เพื่อkṛṣṇa กฺฤษฺณ กฤษณะ คริชณะ จิตสำนึกนี้ จะช่วยให้เจริญขึ้นมาถึงระดับแห่งความรักองค์ bhagavān ภควานฺ ภะคะวาน บฺะกะวาน ที่สูงกว่า ซึ่งจะทำให้เราสมบูรณ์
