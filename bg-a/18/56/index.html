โศลก 56

sarva-karmāṇy api sadā
 สรฺว-กรฺมาณฺยฺ อปิ สทา
 สัรวะ-กัรมาณย อะปิ สะทา
 สารวะ-คารมานิ อพิ สะดา
 kurvāṇo mad-vyapāśrayaḥ
 กุรฺวาโณ มทฺ-วฺยปาศฺรยห์
 กุรวาโณ มัท-วยะปาศระยะห์
 คุรวาโณ มัด-วิยะพาชระยะฮ
 mat-prasādād avāpnoti
 มตฺ-ปฺรสาทาทฺ อวาปฺโนติ
 มัต-ประสาทาท อะวาปโนติ
 มัท-พระสาดาด อวาพโนทิ
 śāśvataṁ padam avyayam
 ศาศฺวตํ ปทมฺ อวฺยยมฺ
 ศาศวะตัง ปะทัม อัวยะยัม
 ชาชวะทัม พะดัม อัพยะยัม

sarva สรฺว สัรวะ สารวะ — ทั้งหมด, karmāṇi กรฺมาณิ กัรมาณิ คารมาณิ — กิจกรรม, api อปิ อะปิ อพิ — ถึงแม้ว่า, sadā สทา สะทา สะดา — เสมอ, kurvāṇaḥ กุรฺวาณห์ กุรวาณะห์ คุรวาณะฮ — ปฏิบัติ, mat-vyapāśrayaḥ มตฺ-วฺยปาศฺรยห์ มัต-วยะปาศระยะห์ มัทฺ-วิยะพาชระยะฮ — ภายใต้การปกป้องของข้า, mat-prasādāt มตฺ-ปฺรสาทาตฺ มัต-ประสาทาต มัท-พระสาดาท — ด้วยความเมตตาของ ข้า, avāpnoti อวาปฺโนติ อะวาปโนติ อวาพโนทิ — เขาบรรลุ, śāśvatam ศาศฺวตมฺ ศาศวะตัม ชาชวะทัม — ความเป็นอมตะ, padam ปทมฺ ปะทัม พะดัม — พระตำหนัก, avyayam อวฺยยมฺ อัวยะยัม อัพยะยัม — ไม่มีวันสูญสลาย

คำแปล

แม้ปฏิบัติในกิจกรรมต่าง ๆ ทั้งหลาย สาวกผู้บริสุทธิ์ ภายใต้การปกป้องของข้าบรรลุถึงพระตำหนักอมตะและไม่มีวันสูญสลาย ด้วยพระกรุณาธิคุณของข้า

คำอธิบาย

คำว่า mad-vyapāśrayaḥ มทฺ-วฺยปาศฺรยห์ มัท-วยะปาศระยะห์ มัด - วิยะพาชระยะฮ หมายถึงภายใต้การปกป้องขององค์ bhagavān ภควานฺ ภะคะวาน บฺะกะวาน การเป็นอิสระจากมลทินทางวัตถุ สาวกผู้บริสุทธิ์ปฏิบัติภายใต้การกำกับของ bhagavān ภควานฺ ภะคะวาน บฺะกะวานหรือพระอาจารย์ทิพย์ซึ่งเป็นผู้แทนของพระองค์ ไม่มีการจำกัดเวลาสำหรับสาวกผู้บริสุทธิ์ ท่านปฏิบัติในกิจกรรมภายใต้การกำกับขององค์ bhagavān ภควานฺ ภะคะวาน บฺะกะวาน วันละยี่สิบสี่ชั่วโมงร้อยเปอร์เซ็นต์ สำหรับสาวกผู้ปฏิบัติใน kṛṣṇa กฺฤษฺณ กฤษณะ คริชณะ จิตสำนึกเช่นนี้พระองค์ทรงมีพระเมตตามาก ถึงแม้จะมีความยากลำบากนานัปการ ท่านจะถูกนำพาไปยังพระตำหนักทิพย์หรือ kṛṣṇaloka กฺฤษฺณโลก กฤษณะโลกะ คริชณะโลคะ ในอนาคตซึ่งรับประกันได้โดยไม่ต้องสงสัย ณ พระตำหนักสูงสุดนี้ไม่มีการเปลี่ยนแปลง ทุกสิ่งทุกอย่างเป็นอมตะ ไม่มีการสูญสลาย และเปี่ยมไปด้วยความรู้
