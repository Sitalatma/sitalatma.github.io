โศลก 11

kāyena manasā buddhyā
 กาเยน มนสา พุทฺธฺยา
 กาเยนะ มะนะสา พุทธยา
 คาเยนะ มะนะสา บุดดฺยา
 kevalair indriyair api
 เกวไลรฺ อินฺทฺริไยรฺ อปิ
 เกวะไลร อินทริไยร อะปิ
 เควะไลร อิรดริไยร อพิฺ
 yoginaḥ karma kurvanti
 โยคินห์ กรฺม กุรฺวนฺติ
 โยคินะห์ กัรมะ กุรวันติ
 โยกินะฮ คารมะ คุรวันทิ
 saṅgaṁ tyaktvātma-śuddhaye
 สงฺคํ ตฺยกฺตฺวาตฺม-ศุทฺธเย
 สังคัง ตยักตวาตมะ-ศุทธะเย
 สังกัม ทยัคทวาทมะ-ชุดดฺะเยฺ

kāyena กาเยน กาเยนะ คาเยนะ — ด้วยร่างกาย, manasā มนสา มะนะสา มะนะสา — ด้วยจิตใจ, buddhyā พุทฺธฺยา พุทธยา บุดดฺยา — ด้วยปัญญา, kevalaiḥ เกวไลห์ เกวะไลห์ เควะไลฮ — บริสุทธิ์, indriyaiḥ อินฺทฺริไยห์ อินทริไยห์ อินดริไยฮ — ด้วยประสาทสัมผัส, api อปิ อะปิ อพิ — แม้แต่, yoginaḥ โยคินห์ โยคินะห์ โยกินะฮ — บุคคลผู้มี kṛṣṇa กฺฤษฺณ กฤษณะ คริชณะ จิตสำนึก, karma กรฺม กัรมะ คารมะ — การกระทำ, kurvanti กุรฺวนฺติ กุรวันติ คุรวันทิ — พวกเขาปฏิบัติ, saṅgam สงฺคมฺ สังคัม สังกัม — การยึดติด, tyaktvā ตฺยกฺตฺวา ตยักตวา ทยัคทวา — สละ, ātma อาตฺม อาตมะ อาทมะ — ของตนเอง, śuddhaye ศุทฺธเย ศุทธะเย ชุดดฺะเย — เพื่อจุดมุ่งหมายแห่งความบริสุทธิ์

คำแปล

yogī โยคี โยคี โยคี  ผู้สละความยึดติด  ปฏิบัติด้วยร่างกาย  จิตใจ  ปัญญา  และแม้แต่ด้วยประสาทสัมผัส  เพียงเพื่อจุดมุ่งหมายแห่งความบริสุทธิ์เท่านั้น

คำอธิบาย

เมื่อเราปฏิบัติ  kṛṣṇa กฺฤษฺณ กฤษณะ คริชณะ  จิตสำนึกโดยสนองประสาทสัมผัสของ  kṛṣṇa กฺฤษฺณ กฤษณะ คริชณะ  เพื่อให้พระองค์ทรงพอพระทัย  การปฏิบัติไม่ว่าด้วยส่วนไหน  เช่น  ร่างกาย  จิตใจ  ปัญญา  หรือแม้แต่ประสาทสัมผัสจะทำให้บริสุทธิ์จากมลทินทางวัตถุ  และจะไม่มีวิบากกรรมทางวัตถุใด  ๆ  ที่เป็นผลมาจากกิจกรรมของผู้มี  kṛṣṇa กฺฤษฺณ กฤษณะ คริชณะ  จิตสำนึก  ฉะนั้น  กิจกรรมบริสุทธิ์โดยทั่วไปเรียกว่า  sad-ācāra สทฺ-อาจาร สัท-อาจาระ -zzz  สามารถปฏิบัติได้โดยง่ายด้วยการปฏิบัติใน  kṛṣṇa กฺฤษฺณ กฤษณะ คริชณะจิตสำนึก  śrī ศฺรี ศรี ชรี  rūpa รูป รูปะ รูพะ  gosvāmī โคสฺวามี โคสวามี โกสวามี  ใน  bhakti-rasāmṛta-sindhu ภกฺติ-รสามฺฤต-สินฺธุ ภักติ-ระสามฤตะ-สินธุ บัฺคธิ--  (1.2.187)  ได้อธิบายไว้ดังนี้

īhā yasya harer dāsye
 อีหา ยสฺย หเรรฺ ทาเสฺย
 อีหา ยัสยะ หะเรร ทาสเย
 อีฮา ยัสยะ ฮะเรร ดาสเย
 karmaṇā manasā girā
 กรฺมณา มนสา คิรา
 กัรมะณา มะนะสา คิรา
 คารมะณา มะนะสา กิรา
 nikhilāsv apy avasthāsu
 นิขิลาสฺวฺ อปฺยฺ อวสฺถาสุ
 นิขิลาสว อัปย อะวัสถาสุ
 นิคิฺลาสุ อพิ อวัสทฺาสุ
 jīvan-muktaḥ sa ucyate
 ชีวนฺ-มุกฺตห์ ส อุจฺยเต
 ชีวัน-มุกตะห์ สะ อุจยะเต
 จีวัน-มุคทะฮ สะ อุชยะเท

“บุคคลผู้ปฏิบัติใน  kṛṣṇa กฺฤษฺณ กฤษณะ คริชณะ  จิตสำนึก  (หรืออีกนัยหนึ่งรับใช้  kṛṣṇa กฺฤษฺณ กฤษณะ คริชณะ  )  ด้วยร่างกาย  จิตใจปัญญา  และคำพูดเป็นผู้หลุดพ้นแม้อยู่ในโลกวัตถุ  ถึงแม้ท่านอาจปฏิบัติสิ่งที่เรียกว่ากิจกรรมทางวัตถุมากมาย”  แต่ไม่มีอหังการเพราะท่านไม่เชื่อว่าตัวท่านคือร่างวัตถุนี้หรือว่าตัวท่านเป็นเจ้าของร่างวัตถุ  รู้ดีว่าตัวท่านไม่ใช่ร่างกายนี้และร่างกายนี้ไม่ใช่เป็นของท่าน  ตัวท่านเองเป็นของ  kṛṣṇa กฺฤษฺณ กฤษณะ คริชณะ  และร่างกายก็เช่นกันเป็นของ  kṛṣṇa กฺฤษฺณ กฤษณะ คริชณะ  เมื่อนำทุกสิ่งทุกอย่างที่ผลิตโดย  ร่างกาย  จิตใจ  ปัญญา  คำพูด  ชีวิต  ทรัพย์สมบัติ  ฯลฯ  และอะไรก็แล้วแต่ที่ท่านอาจเป็นเจ้าของ  นำมารับใช้  kṛṣṇa กฺฤษฺณ กฤษณะ คริชณะ  เช่นนี้  ท่านจะประสานกับ  kṛṣṇa กฺฤษฺณ กฤษณะ คริชณะ  ทันทีโดยเป็นหนึ่งเดียวกับ  kṛṣṇa กฺฤษฺณ กฤษณะ คริชณะ  และปราศจากอหังการที่จะนำท่านให้เชื่อว่าตัวท่านคือร่างกาย  ฯลฯ  นี่คือระดับสมบูรณ์แห่ง  kṛṣṇa กฺฤษฺณ กฤษณะ คริชณะ  จิตสำนึก
