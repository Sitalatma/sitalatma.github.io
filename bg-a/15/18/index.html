บทที่ สิบห้า

โยค แห่งองค์ภควาน

โศลก 18

yasmāt kṣaram atīto ’ham
 ยสฺมาตฺ กฺษรมฺ อตีโต ’หมฺ
 ยัสมาต กษะรัม อะตีโต ’หัม
 ยัสมาท คชะรัม อทีโท ่ฮัม
 akṣarād api cottamaḥ
 อกฺษราทฺ อปิ โจตฺตมห์
 อักษะราท อะปิ โจตตะมะห์
 อัคชะราด อพิ โชททะมะฮ
 ato ’smi loke vede ca
 อโต ’สฺมิ โลเก เวเท จ
 อะโต ’สมิ โลเก เวเท จะ
 อโท ่สมิ โลเค เวเด ชะ
 prathitaḥ puruṣottamaḥ
 ปฺรถิตห์ ปุรุโษตฺตมห์
 ประถิตะห์ ปุรุโษตตะมะห์
 พระทิฺทะฮ พุรุโชททะมะฮ

yasmāt ยสฺมาตฺ ยัสมาต ยัสมาท — เพราะว่า,kṣaram กฺษรมฺ กษะรัม คชะรัม — ผู้ผิดพลาด, atītaḥ อตีตห์ อะตีตะห์ อทีทะฮ — เป็นทิพย์, aham อหมฺ อะหัม อฮัม — ข้าเป็น, akṣarāt อกฺษราตฺ อักษะราต อัคชะราท — เหนือผู้ไม่ผิดพลาด, api อปิ อะปิ อพิ — เช่นกัน, ca จ จะ ชะ — และ, uttamaḥ อุตฺตมห์ อุตตะมะห์ อุททะมะฮ — ดีที่สุด, ataḥ อตห์ อะตะห์ อทะฮ — ดังนั้น, asmi อสฺมิ อัสมิ อัสมิ — ข้าเป็น, loke โลเก โลเก โลเค — ในโลก, vede เวเท เวเท เวเด — ในวรรณกรรมพระ veda เวท เวทะ เวท, ca จ จะ ชะ — และ, prathitaḥ ปฺรถิตห์ ประถิตะห์ พระทิฺทะฮ — มีชื่อเสียง, puruṣa-uttamaḥ ปุรุษ-อุตฺตมห์ ปุรุษะ-อุตตะมะห์ พุรุชะ-อุททะมะฮ — ในฐานะบุคลิกภาพสูงสุด

คำแปล

เพราะว่าข้าเป็นทิพย์อยู่เหนือทั้งผู้ผิดพลาดและผู้ไม่ผิดพลาด และเนื่องจากข้ายิ่งใหญ่ที่สุด ข้าจึงมีชื่อเสียงทั้งในโลกและในคัมภีร์พระ veda เวท เวทะ เวท ในฐานะที่เป็นองค์ภควาน

คำอธิบาย

ไม่มีผู้ใดมีความสามารถเกินไปกว่า kṛṣṇa กฺฤษฺณ กฤษณะ คริชณะ บุคลิกภาพสูงสุดแห่งพระเจ้า ไม่ว่าพันธวิญญาณหรืออิสรวิญญาณ ดังนั้น พระองค์ทรงเป็นบุคลิกภาพผู้ยิ่งใหญ่ที่สุดณ ที่นี้ ได้กล่าวไว้อย่างชัดเจนว่า ทั้งสิ่งมีชีวิตและองค์ bhagavān ภควานฺ ภะคะวาน บฺะกะวาน เป็นปัจเจกบุคคล ข้อแตกต่างคือสิ่งมีชีวิตไม่ว่าอยู่ในระดับที่ถูกพันธนาการหรือในระดับที่มีอิสรภาพ ไม่สามารถมีปริมาณเหนือกว่าพลังอำนาจที่ไม่สามารถมองเห็นได้ของบุคลิกภาพสูงสุดแห่งพระเจ้าจึงเป็นการไม่ถูกต้องที่คิดว่าองค์ bhagavān ภควานฺ ภะคะวาน บฺะกะวาน และสิ่งมีชีวิตอยู่ในระดับเดียวกัน หรือเท่าเทียมกันในทุก ๆ ด้าน จะมีคำถามเกี่ยวกับความสูงกว่าและต่ำกว่าระหว่างบุคลิกภาพเหล่านี้เสมอ คำว่า uttama อุตฺตม อุตตะมะ อุททะมะ มีความสำคัญมาก ไม่มีผู้ใดสามารถอยู่เหนือองค์ bhagavān ภควานฺ ภะคะวาน บฺะกะวาน 

คำว่า loke โลเก โลเก โลเค แสดงถึง “ใน pauruṣa āgama เปารุษ อาคม เปารุษะ อาคะมะ โพรุชะ อากะมะ (พระคัมภีร์ smṛti สฺมฺฤติ สมฤติ สมริทิ )” ดังที่ยืนยันไว้ในพจนานุกรม nirukti นิรุกฺติ นิรุกติ นิรุคทิ ว่า lokyate vedārtho ’nena โลกฺยเต เวทารฺโถ เนน โลกยะเต เวทารโถ เนนะ โลคยะเท เวดารโทฺ ’เนนะ “จุดมุ่งหมายของคัมภีร์พระ veda เวท เวทะ เวท พระคัมภีร์ smṛti สฺมฺฤติ สมฤติ สมริทิ ได้อธิบายไว้”

องค์ bhagavān ภควานฺ ภะคะวาน บฺะกะวาน ในรูปลักษณ์ paramātmā ปรมาตฺมา ปะระมาตมา พะระมาทมา ภายในหัวใจทุกคน ได้อธิบายไว้ในคัมภีร์พระ veda เวท เวทะ เวท เช่นกัน โศลกเหล่านี้ปรากฏในคัมภีร์พระ veda เวท เวทะ เวท (chāndogya upaniṣad ฉานฺโทคฺย อุปนิษทฺ ฉานโทคยะ อุปะนิษัท ชัฺนโดกยะ อุพะนิชัด 8.12.3) tāvad eṣa samprasādo ’smāc charīrāt samutthāya paraṁ jyoti-rūpaṁ sampadya svena rūpeṇābhiniṣpadyate sa uttamaḥ puruṣaḥ ตาวทฺ เอษ สมฺปฺรสาโท สฺมาจฺ ฉรีราตฺ สมุตฺถาย ปรํ โชฺยติ-รูปํ สมฺปทฺย เสฺวน รูเปณาภินิษฺปทฺยเต ส อุตฺตมห์ ปุรุษห์ ตาวัท เอษะ สัมประสาโท สมาจ ฉะรีราต สะมุตถายะ ปะรัง ชโยติ-รูปัง สัมปัทยะ สเวนะ รูเปณาภินิษปัทยะเต สะ อุตตะมะห์ ปุรุษะห์ ทาวัด เอชะ สัมพระสาโด ’สมาช ชฺะรีราท สะมุททฺายะ พะรัม จโยทิ-รูพัม สัมพัดยะ สเวนะ รูเพณาบิฺนิชพัดยะเท สะ อุททะมะฮ พุรุชะฮ “องค์อภิวิญญาณที่ทรงออกมาจากร่างกายแล้วจึงเสด็จเข้าไปใน brahma-jyotir พฺรหฺม-โชฺยติรฺ พรัหมะ-ชโยติร บระฮมะจโยทิ ที่ไร้รูปลักษณ์ จากนั้นด้วยรูปลักษณ์ของพระองค์ พระองค์ทรงไว้ซึ่งบุคลิกลักษณะทิพย์ bhagavān ภควานฺ ภะคะวาน บฺะกะวาน องค์นั้นเรียกว่าบุคลิกภาพสูงสุด” เช่นนี้หมายความว่า บุคลิกภาพสูงสุดทรงแสดงและทรงแพร่กระจายรัศมีทิพย์ของพระองค์ซึ่งเป็นแสงอันเจิดจรัสสูงสุด บุคลิกภาพสูงสุดพระองค์นั้นทรงมีรูปลักษณ์อยู่ภายในหัวใจของทุก ๆ คนด้วยเช่นกันในรูปของ paramātmā ปรมาตฺมา ปะระมาตมา พะระมาทมา จากการอวตารมาเป็นบุตรของ satyavatī สตฺยวตี สัตยะวะตี สัทยะวะที และ parāśara ปราศร ปะราศะระ พะราชะระ องค์ vyāsadeva วฺยาสเทว วยาสะเทวะ วิยาสะเดวะ ทรงอธิบายความรู้พระ veda เวท เวทะ เวท
