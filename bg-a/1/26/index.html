ภควัต-คีตา ฉบับเดิม

บทที่ หนึ่ง

สำรวจกองทัพที่สมรภูมิ กุรุกฺเษตฺร

โศลก 26

tatrāpaśyat sthitān pārthaḥ
 ตตฺราปศฺยตฺ สฺถิตานฺ ปารฺถห์
 ตัตราปัศยัต สถิตาน ปารถะห์
 ทะทราพัชยัท สทิฺทาน พารทฺะฮ
 pitṝn atha pitāmahān
 ปิตฺฤๅนฺ อถ ปิตามหานฺ
 ปิตฤๅน อะถะ ปิตามะหาน
 พิทรีน อทฺะ พิทามะฮาน
 ācāryān mātulān bhrātṝn
 อาจารฺยานฺ มาตุลานฺ ภฺราตฺฤๅนฺ
 อาจารยาน มาตุลาน ภราตฤๅน
 อาชารยาน มาทุลาน บฺราทรีน
 putrān pautrān sakhīṁs tathā
 ปุตฺรานฺ เปาตฺรานฺ สขีํสฺ ตถา
 ปุตราน เปาตราน สะขีงส ตะถา
 พุทราน โพทราน สะคีฺมส ทะทฺา
 śvaśurān suhṛdaś caiva
 ศฺวศุรานฺ สุหฺฤทศฺ ไจว
 ศวะศุราน สุหฤทัศ ไจวะ
 ชวะชุราน สุฮริดัช ไชวะ
 senayor ubhayor api
 เสนโยรฺ อุภโยรฺ อปิ
 เสนะโยร อุภะโยร อะปิ
 เสนะโยร อุบฺะโยร อพิฺ

tatra ตตฺร ตัตระ ทะทระ — ที่นั่น, apaśyat อปศฺยตฺ อะปัศยัต อพัชยัท — ทรงเห็น, sthitān สฺถิตานฺ สถิตาน สทิฺทาน — ยืนอยู่, pārthaḥ ปารฺถห์ ปารถะห์ พารทฺะฮ — arjuna อรฺชุน อัรชุนะ อารจุนะ, pitṝn ปิตฺฤๅนฺ ปิตฤๅน พิทรีน — พระบิดา, atha อถ อะถะ อทฺะ — เช่นกัน, pitāmahān ปิตามหานฺ ปิตามะหาน พิทามะฮาน — พระอัยกา, ācāryān อาจารฺยานฺ อาจารยาน อาชารยาน — พระอาจารย์, mātulān มาตุลานฺ มาตุลาน มาทุลาน — พระมาตุลา, bhrātṝn ภฺราตฺฤๅนฺ ภราตฤๅน บฺราทรีน — พี่น้อง, putrān ปุตฺรานฺ ปุตราน พุทราน — บุตร, pautrān เปาตฺรานฺ เปาตราน โพทราน — หลาน, sakhīn สขีนฺ สะขีน สะคีฺน — เพื่อน, tathā ตถา ตะถา ทะทฺา — ด้วยเหมือน กัน, śvaśurān ศฺวศุรานฺ ศวะศุราน ชวะชุราน — พระสัสสุระ, suhṛdaḥ สุหฺฤทห์ สุหฤทะห์ สุฮริดะฮ — ผู้ปรารถนาดี, ca จ จะ ชะ — เช่นกัน, eva เอว เอวะ เอวะ — แน่นอน, senayoḥ เสนโยห์ เสนะโยห์ เสนะโยฮ — ของกองทัพ, ubhayoḥ อุภโยห์ อุภะโยห์ อุบฺะโยฮ — ของทั้งสองฝ่าย, api อปิ อะปิ อพิ — รวม

คำแปล

ท่ามกลางกองทัพทั้งสองฝ่าย arjuna อรฺชุน อัรชุนะ อารจุนะ ทรงเห็น พระบิดา พระอัยกา พระอาจารย์พระมาตุลา พระเชษฐา พระอนุชา พระโอรส พระราชนัดดา พระสหาย รวมทั้งพระสัสสุระและผู้ปรารถนาดีอื่น ๆ

คำอธิบาย

ที่สมรภูมิ arjuna อรฺชุน อัรชุนะ อารจุนะ ทรงเห็นญาติ ๆ ทั้งหลาย บุคคลเช่น bhūriśravā ภูริศฺรวา ภูริศระวา บํูริชระว ารุ่นเดียวกับพระบิดา พระอัยกา bhīṣma ภีษฺม ภีษมะ บีฺชมะ และ somadatta โสมทตฺต โสมะทัตตะ โสมะดัททะ พระอาจารย์ เช่น droṇācārya โทฺรณาจารฺย ทโรณาจารยะ โดรณาชารยะ และ คริพา-ชารยะ พระมาตุลา śalya ศลฺย ศัลยะ ชัลยะ และ śakuni ศกุนิ ศะกุนิ ชะคุนิ พระเชษฐา พระอนุชา เช่น duryodhana ทุโรฺยธน ทุรโยธะนะ ดุรโยดฺะนะ พระโอรสเช่น lakṣmaṇa ลกฺษฺมณ ลักษมะณะ ลัคชมะณะ สหาย เช่น aśvatthāmā อศฺวตฺถามา อัศวัตถามา อัชวัททฺม และผู้ปรารถดี เช่น kṛtavarmā กฺฤตวรฺมา กฤตะวัรมา คริทะวารมา ฯลฯ arjuna อรฺชุน อัรชุนะ อารจุนะทรงเห็นสหายเป็นจำนวนมากอยู่ในกองทัพทั้งสองฝ่าย
