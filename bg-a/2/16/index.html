ภควัต-คีตา ฉบับเดิม

บทที่ สอง

บทสรุป ภควทฺ-คีตา

โศลก 16

nāsato vidyate bhāvo
 นาสโต วิทฺยเต ภาโว
 นาสะโต วิทยะเต ภาโว
 นาสะโท วิดยะเท บฺาโว
 nābhāvo vidyate sataḥ
 นาภาโว วิทฺยเต สตห์
 นาภาโว วิทยะเต สะตะห์
 นาบฺาโว วิดยะเท สะทะฮ
 ubhayor api dṛṣṭo ’ntas
 อุภโยรฺ อปิ ทฺฤษฺโฏ ’นฺตสฺ
 อุภะโยร อะปิ ทฤษโฏ ’นตัส
 อุบฺะโยร อพิ ดริชโท ´นทัส
 tv anayos tattva-darśibhiḥ
 ตฺวฺ อนโยสฺ ตตฺตฺว-ทรฺศิภิห์
 ตว อะนะโยส ตัตตวะ-ทัรศิภิห์
 ทุ อนะโยส ทัทวะ-ดารชิบิฮ

na น นะ นะ — ไม่เคย, asataḥ อสตห์ อะสะตะห์ อสะทะฮ — แห่งการไม่มีอยู่, vidyate วิทฺยเต วิทยะเต วิดยะเท — มี, bhāvaḥ ภาวห์ ภาวะห์ บฺาวะฮ — ความทนทาน, na น นะ นะ — ไม่เคย, abhāvaḥ อภาวห์ อะภาวะห์ อบาวะฮ — คุณภาพที่เปลี่ยนแปลง, vidyate วิทฺยเต วิทยะเต วิดยะเท — มี, sataḥ สตห์ สะตะห์ สะทะฮ — แห่งอมตะ, ubhayoḥ อุภโยห์ อุภะโยห์ อุบฺะโยฮ — ของทั้ง สอง, api อปิ อะปิ อพิ — เป็นความจริง, dṛṣṭaḥ ทฺฤษฺฏห์ ทฤษฏะห์ ดริชทะฮ — สังเกตดู, antaḥ อนฺตห์ อันตะห์ อันทะฮ — สรุป, tu ตุ ตุ ทุ — แน่นอน, anayoḥ อนโยห์ อะนะโยห์ อนะโยฮ — ของ เขาทั้งหลาย, tattva ตตฺตฺว ตัตตวะ ทัททวะ — แห่งความจริง, darśibhiḥ ทรฺศิภิห์ ทัรศิภิห์ ดารชิบิฮ — โดยผู้เห็น

คำแปล

ผู้เห็นสัจธรรมได้สรุปไว้ว่า สำหรับสิ่งที่ไม่มีอยู่จริง (ร่างกายวัตถุ) จะไม่มีความคงทนถาวร และสำหรับสิ่งที่เป็นอมตะ (ดวงวิญญาณ) จะไม่มีการเปลี่ยนแปลงพวกเขาสรุปเช่นนี้จากการศึกษาธรรมชาติของทั้งสองสิ่ง

คำอธิบาย

ร่างกายที่เปลี่ยนแปลงไม่มีความคงทนถาวร เนื่องจากร่างกายเปลี่ยนแปลงอยู่ตลอดเวลาจากการกระทำและผลการกระทำของเซลส์ต่างๆ ดังที่วิทยาศาสตร์การแพทย์สมัยปัจจุบันยอมรับ ดังนั้น การเจริญเติบโตและความแก่ชราจึงเกิดขึ้นกับร่างกาย แต่ดวงวิญญาณนั้นจะคงความเป็นอมตะนิรันดร แม้ร่างกายและจิตใจจะเปลี่ยนไปทั้งหมด นี่คือข้อแตกต่างระหว่างวัตถุและดวงวิญญาณ โดยธรรมชาติร่างกายเปลี่ยนแปลงอยู่ตลอดเวลา แต่ดวงวิญญาณเป็นอมตะ ผลสรุปเช่นนี้ผู้พบเห็นสัจธรรมในทุกระดับได้ยืนยันไว้แล้ว ทั้งผู้ไม่เชื่อในรูปลักษณ์และผู้ที่เชื่อในรูปลักษณ์ ใน viṣṇu purāṇa วิษฺณุ ปุราณ วิษณุ ปุราณะ วิชณุ พุราณะ (2.12.38) กล่าวไว้ว่า องค์ viṣṇu วิษฺณุ วิษณุ วิชณุ และที่ประทับของพระองค์มีรัศมีทิพย์ในตัวเอง(jyotīṁṣi viṣṇur bhuvanāni viṣṇuḥ โชฺยตีํษิ วิษฺณุรฺ ภุวนานิ วิษฺณุห์ ชโยตีงษิ วิษณุร ภุวะนานิ วิษณุห์ จโยทีมชิ วิชณุร บํุวะนานิ วิชณุฮ ) คำว่า มีอยู่จริงและไม่มีอยู่จริงหมายถึงดวงวิญญาณและวัตถุเท่านั้น นั่นคือคำบอกเล่าของผู้ที่เห็นสัจธรรมทั้งหลาย

นี่คือจุดเริ่มต้นของคำสอนโดยองค์ bhagavān ภควานฺ ภะคะวาน บฺะกะวาน ที่ให้แก่สิ่งมีชีวิตผู้มีความสับสนมืดมนอยู่ภายใต้อิทธิพลของอวิชชา การขจัดอวิชชาจะต้องมีการสถาปนาความสัมพันธ์ที่เป็นอมตะระหว่างผู้บูชาและผู้รับการบูชา จากนั้น จะต้องมีความเข้าใจข้อแตกต่างระหว่างสิ่งมีชีวิตอันเป็นละอองอณูและบุคลิกภาพสูงสุดแห่งพระเจ้า เราสามารถเข้าใจธรรมชาติขององค์ bhagavān ภควานฺ ภะคะวาน บฺะกะวาน ด้วยการศึกษาตนเองอย่างละเอียดถี่ถ้วน ข้อแตกต่างระหว่างตัวเราและองค์ bhagavān ภควานฺ ภะคะวาน บฺะกะวาน คือ ความสัมพันธ์ระหว่างเศษย่อยกับส่วนรวมทั้งหมด ทั้งใน vedānta-sūtra เวทานฺต-สูตฺร เวทานตะ-สูตระ เวดานธะ - สูทระ และ śrīmad-bhāgavatam ศฺรีมทฺ-ภาควตมฺ ศรีมัท-ภาคะวะตัม ชรีมัด - บฺากะวะธัม องค์ bhagavān ภควานฺ ภะคะวาน บฺะกะวาน ทรงได้รับการยอมรับว่าเป็นแหล่งกำเนิดของสิ่งที่ปรากฎมาทั้งหมด การปรากฎเช่นนี้ทั้งธรรมชาติระดับสูงและธรรมชาติระดับต่ำมีประสบการณ์ สิ่งมีชีวิตอยู่ในธรรมชาติที่สูงกว่าดังจะเปิดเผยในบทที่เจ็ด แม้ไม่มีข้อแตกต่างระหว่างพลังงานและแหล่งกำเนิดพลังงาน แหล่งกำเนิดพลังงานได้รับการยอมรับว่าคือองค์ bhagavān ภควานฺ ภะคะวาน บฺะกะวาน และพลังงานหรือธรรมชาติได้รับการยอมรับว่าเป็นส่วนที่รองลงมา ฉะนั้น สิ่งมีชีวิตจะเป็นรององค์ bhagavān ภควานฺ ภะคะวาน บฺะกะวาน เสมอ ดังเช่นเจ้านายและบ่าวหรืออาจารย์และศิษย์ ความรู้ที่ชัดเจนเช่นนี้ไม่สามารถเข้าใจได้สำหรับผู้ที่อยู่ภายใต้มนต์สะกดของอวิชชา เพื่อขจัดอวิชชานี้องค์ bhagavān ภควานฺ ภะคะวาน บฺะกะวาน ทรงสอน bhagavad-gītā ภควทฺ-คีตา ภะคะวัท-คีตา ดฺยานะ-โยกะ เพื่อให้สิ่งมีชีวิตทั้งหลายได้รู้แจ้งสัจธรรมอันแท้จริงตลอดกาล
