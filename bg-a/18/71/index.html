โศลก 71

śraddhāvān anasūyaś ca
 ศฺรทฺธาวานฺ อนสูยศฺ จ
 ศรัทธาวาน อะนะสูยัศ จะ
 ชรัดดฺาวาน อนะสูยัช ชะ
 śṛṇuyād api yo naraḥ
 ศฺฤณุยาทฺ อปิ โย นรห์
 ศฤณุยาท อะปิ โย นะระห์
 ชริณุยาด อพิ โย นะระฮ
 so ’pi muktaḥ śubhāḻ lokān
 โส ’ปิ มุกฺตห์ ศุภาฬฺ โลกานฺ
 โส ’ปิ มุกตะห์ ศุภาฬ โลกาน
 โส ่พิ มุคทะฮ ชูบฺาล โลคาน
 prāpnuyāt puṇya-karmaṇām
 ปฺราปฺนุยาตฺ ปุณฺย-กรฺมณามฺ
 ปราปนุยาต ปุณยะ-กัรมะณาม
 พราพนุยาท พุณยะ-คารมะณาม

śraddhā-vān ศฺรทฺธา-วานฺ ศรัทธา-วาน ชรัดดฺา-วาน — ศรัทธา, anasūyaḥ อนสูยห์ อะนะสูยะห์ อนะสูยะฮ — ไม่อิจฉาริษยา, ca จ จะ ชะ — และ, śṛṇuyāt ศฺฤณุยาตฺ ศฤณุยาต ชริณุยาท — สดับฟัง, api อปิ อะปิ อพิ — แน่นอน, yaḥ ยห์ ยะห์ ยะฮ — เขา, naraḥ นรห์ นะระห์ นะระฮ — มนุษย์, saḥ สห์ สะห์ สะฮ — เขา, api อปิ อะปิ อพิ — เช่นกัน, muktaḥ มุกฺตห์ มุกตะห์ มุคทะฮ — หลุดพ้น, śubhān ศุภานฺ ศุภาน ชุบฺาน — เป็นมงคล, lokān โลกานฺ โลกาน โลคาน — โลก, prāpnuyāt ปฺราปฺนุยาตฺ ปราปนุยาต พราพนุยาท — เขาบรรลุ, puṇya-karmaṇām ปุณฺย-กรฺมณามฺ ปุณยะ-กัรมะณาม พุณยะ-คารมะณาม — แห่งบุญ

คำแปล

และผู้ที่สดับฟังด้วยความศรัทธา โดยปราศจากความอิจฉาริษยา จะเป็นอิสระจากผลบาป และบรรลุถึงโลกอันเป็นสิริมงคลที่คนมีบุญบารมีพำนักอาศัยอยู่

คำอธิบาย

โศลกที่หกสิบเจ็ดของบทนี้ องค์ bhagavān ภควานฺ ภะคะวาน บฺะกะวาน ทรงห้ามไว้อย่างชัดเจนว่าไม่ควรสอน แก่พวกที่อิจฉาริษยาพระองค์ อีกนัยหนึ่ง bhagavad-gītā ภควทฺ-คีตา ภะคะวัท-คีตา ดฺยานะ-โยกะ มีไว้สำหรับสาวกเท่านั้น แต่บางครั้งสาวกขององค์ bhagavān ภควานฺ ภะคะวาน บฺะกะวาน เปิดชั้นเรียน และในชั้นเรียนไม่ใช่ว่านักศึกษาทุกคนเป็นสาวก คำถามคือแล้วทำไมจึงเปิดชั้นเรียน? อธิบาย ณ ที่นี้ว่าถึงแม้ว่าไม่ใช่ทุกคนเป็นสาวก แต่มีหลายคนที่ไม่อิจฉาริษยา kṛṣṇa กฺฤษฺณ กฤษณะ คริชณะ และมีความศรัทธาพระองค์ในฐานะที่เป็นบุคลิกภาพสูงสุดแห่งพระเจ้า หากบุคคลเหล่านี้สดับฟังเกี่ยวกับองค์ bhagavān ภควานฺ ภะคะวาน บฺะกะวาน จากสาวกผู้ที่เชื่อถือได้ ผลก็คือพวกเขาจะเป็นอิสระจากผลบาปทั้งปวงทันที และหลังจากนั้นจะบรรลุถึงระบบดาวเคราะห์ที่คนมีคุณธรรมทั้งหลายอาศัยอยู่ ดังนั้น เพียงแต่สดับฟัง bhagavad-gītā ภควทฺ-คีตา ภะคะวัท-คีตา ดฺยานะ-โยกะ แม้บุคคลผู้ไม่พยายามเป็นสาวกผู้บริสุทธิ์จะบรรลุถึงผลแห่งกิจกรรมที่มีคุณธรรม ดังนั้นสาวกผู้บริสุทธิ์ของพระองค์เปิดโอกาสให้ทุก ๆ คนเป็นอิสระจากผลบาปทั้งหมดและมาเป็นสาวกขององค์ bhagavān ภควานฺ ภะคะวาน บฺะกะวาน 

โดยทั่วไปผู้ที่เป็นอิสระจากผลบาป พวกที่มีคุณธรรมจะปฏิบัติ kṛṣṇa กฺฤษฺณ กฤษณะ คริชณะ จิตสำนึกได้โดยง่ายดาย คำว่า puṇya-karmaṇām ปุณฺย-กรฺมณามฺ ปุณยะ-กัรมะณาม พุณยะ-คารมะณาม มีความสำคัญมาก ณ ที่นี้ ซึ่งหมายถึงการปฏิบัติพิธีบูชาที่ยิ่งใหญ่เช่น aśvamedha-yajña อศฺวเมธ-ยชฺญ อัศวะเมธะ-ยัชญะ อัชวะเมดฺะ-ยะกยะ ที่ได้กล่าวไว้ในวรรณกรรมพระ veda เวท เวทะ เวท พวกที่มีคุณธรรมปฏิบัติการอุทิศตนเสียสละรับใช้ แต่ยังไม่บริสุทธิ์ทีเดียวสามารถบรรลุถึงระบบดาวเคราะห์ของดาวเหนือหรือ dhruvaloka ธฺรุวโลก ธรุวะโลกะ ดฺรุวะโลคะ ที่ dhruva mahārāja ธฺรุว มหาราช ธรุวะ มะหาราชะ ดฺรุวะ มะฮาราจะ สาวกผู้ยิ่งใหญ่ขององค์ bhagavān ภควานฺ ภะคะวาน บฺะกะวาน ทรงประทับอยู่ และทรงมีดาวเคราะห์พิเศษเรียกว่าดาวเหนือ
