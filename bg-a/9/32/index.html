โศลก 32

māṁ hi pārtha vyapāśritya
 มำ หิ ปารฺถ วฺยปาศฺริตฺย
 มาง หิ ปารถะ วยะปาศริตยะ
 มาม ฮิ พารทฺะ วิยะพาชริทยะ
 ye ’pi syuḥ pāpa-yonayaḥ
 เย ’ปิ สฺยุห์ ปาป-โยนยห์
 เย ’ปิ สยุห์ ปาปะ-โยนะยะห์
 เย พิ สยุฮ พาพะ-โยนะยะฮ
 striyo vaiśyās tathā śūdrās
 สฺตฺริโย ไวศฺยาสฺ ตถา ศูทฺราสฺ
 สตริโย ไวศยาส ตะถา ศูทราส
 สทริโย ไวชยาส ทะทฺา ชูดราส
 te ’pi yānti parāṁ gatim
 เต ’ปิ ยานฺติ ปรำ คติมฺ
 เต ’ปิ ยานติ ปะราง คะติม
 เท พิ ยานทิ พะราม กะทิม

mām มามฺ มาม มาม — ของข้า, hi หิ หิ ฮิ — แน่นอน, pārtha ปารฺถ ปารถะ พรทฺะ — โอ้ โอรสพระนาง pṛthā ปฺฤถา ปฤถา พริทฺา, vyapāśritya วฺยปาศฺริตฺย วยะปาศริตยะ วิยะพาชริทยะ — มีที่พึ่งโดย เฉพาะ, ye เย เย เย — พวกเขา, api อปิ อะปิ อพิ — เช่นกัน, syuḥ สฺยุห์ สยุห์ สยุฮ — เป็น, pāpa-yonayaḥ ปาป-โยนยห์ ปาปะ-โยนะยะห์ พาพะ-โยนะยะฮ — เกิดในครอบครัวต่ำ, striyaḥ สฺตฺริยห์ สตริยะห์ สทิรยะฮ-สตรี, vaiśyāḥ ไวศฺยาห์ ไวศยาห์ ไวชยาฮ — พ่อค้า, tathā ตถา ตะถา ทะทฺา — เช่นกัน, śūdrāḥ ศูทฺราห์ ศูทราห์ ชูดราฮ — คนชั้นต่ำ,te api เต อปิ เต อะปิ เท อพิ — แม้พวกเขา, yānti ยานฺติ ยานติ ยานทิ — ไป, parām ปรามฺ ปะราม พะราม — ถึงองค์ภควาน, gatim คติมฺ คะติม กะทิม — จุดมุ่งหมาย

คำแปล

โอ้  โอรสพระนาง  pṛthā ปฺฤถา ปฤถา พริทฺา  พวกที่มาพึ่งข้า  ถึงแม้ว่าเกิดมาต่ำ  เช่นสตรี  vaiśya ไวศฺย ไวศยะ ไวชยะ  (พ่อค้า)  และ  śūdra ศูทฺร ศูทระ ชูดระ  (ผู้ใช้แรงงาน)  สามารถบรรลุถึงจุดมุ่งหมายสูงสุดได้

คำอธิบาย

ณ  ที่นี้องค์  bhagavān ภควานฺ ภะคะวาน บฺะกะวาน  ทรงประกาศอย่างชัดเจนว่า  ในการอุทิศตนเสียสละรับใช้ไม่มีข้อแบ่งแยกระหว่างชนชั้นต่ำและชนชั้นสูง  ในแนวคิดแห่งชีวิตทางวัตถุมีการแบ่งแยกเช่นนี้  แต่สำหรับผู้ปฏิบัติการอุทิศตนเสียสละรับใช้ทิพย์ต่อองค์  bhagavān ภควานฺ ภะคะวาน บฺะกะวาน  จะไม่มี  ทุกคนมีสิทธิ์ที่จะไปถึงจุดมุ่งหมายสูงสุดได้  ใน  śrīmad-bhāgavatam ศฺรีมทฺ-ภาควตมฺ ศรีมัท-ภาคะวะตัม ชรีมัด-  (2.4.18)  กล่าวไว้ว่า  แม้พวกที่ต่ำสุดเรียกว่า  caṇḍāla จณฺฑาล จัณฑาละ ชัณดาละ  (คนกินสุนัข)  สามารถทำให้บริสุทธิ์ขึ้นมาได้ด้วยการมาคบหาสมาคมกับสาวกผู้บริสุทธิ์  ฉะนั้น  การอุทิศตนเสียสละรับใช้และการนำทางของสาวกผู้บริสุทธิ์มีพลังอำนาจมากจนไม่มีการเลือกปฏิบัติระหว่างชนชั้นที่ต่ำกว่าและชนชั้นที่สูงกว่า  ทุก  ๆ  คนนำไปปฏิบัติได้  คนที่เรียบง่ายที่สุดมาพึ่งสาวกผู้บริสุทธิ์สามารถทำให้บริสุทธิ์ขึ้นได้จากการนำทางที่ถูกต้อง  ตามระดับต่าง  ๆ  ของธรรมชาติวัตถุ  มนุษย์แบ่งอยู่ในระดับแห่งความดี  (brāhmaṇa พฺราหฺมณ พราหมะณะ บราฮมะณะ  )  ระดับแห่งตัณหา  (  kṣatriya กฺษตฺริย กษัตริยะ กษัตริย์  หรือผู้บริหาร)ระดับผสมกันระหว่างตัณหาและอวิชชา  (  vaiśya ไวศฺย ไวศยะ ไวชยะ  หรือพ่อค้า)  และระดับแห่งอวิชชา(  śūdra ศูทฺร ศูทระ ชูดระ  หรือผู้ใช้แรงงาน)  พวกที่ต่ำไปกว่านี้เรียกว่า  caṇḍāla จณฺฑาล จัณฑาละ ชัณดาละ  ที่เกิดในครอบครัวบาปโดยทั่วไปการคบหาสมาคมกับผู้ที่เกิดในครอบครัวบาป  ชนชั้นสูงรับไม่ได้  แต่วิธีแห่งการอุทิศตนเสียสละรับใช้มีพลังอำนาจมาก  สาวกผู้บริสุทธิ์ขององค์  bhagavān ภควานฺ ภะคะวาน บฺะกะวาน  สามารถทำให้คนในชั้นต่ำทั้งหมดบรรลุถึงความสมบูรณ์สูงสุดแห่งชีวิตได้  เป็นเช่นนี้ได้เมื่อเ  rāma ราม รามะ ราม  าพึ่งkṛṣṇa กฺฤษฺณ กฤษณะ คริชณะ  เท่านั้น  ดังที่ได้แสดงไว้  ณ  ที่นี้ด้วยคำว่า  vyapāśritya วฺยปาศฺริตฺย วยะปาศริตยะ วิยะพาชริทยะ  เราต้องมาพึ่ง  kṛṣṇa กฺฤษฺณ กฤษณะ คริชณะโดยสมบูรณ์แล้วจะกลายมาเป็นผู้ที่ยิ่งใหญ่กว่า  jñānī ชฺญานี ชญานี กยานี  และ  yoga โยค โยคะ โยกะ  ผู้ยิ่งใหญ่
