โศลก 13-15

idam adya mayā labdham
 อิทมฺ อทฺย มยา ลพฺธมฺ
 อิทัม อัทยะ มะยา ลัพธัม
 อิดัม อัดยะ มะยา ลับดัฺม
 imaṁ prāpsye manoratham
 อิมํ ปฺราปฺเสฺย มโนรถมฺ
 อิมัง ปราปสเย มะโนระถัม
 อิมัม พราพสเย มะโนระทัฺม
 idam astīdam api me
 อิทมฺ อสฺตีทมฺ อปิ เม
 อิทัม อัสตีทัม อะปิ เม
 อิดัม อัสทีดัม อพิ เม
 bhaviṣyati punar dhanam
 ภวิษฺยติ ปุนรฺ ธนมฺ
 ภะวิษยะติ ปุนัร ธะนัม
 บฺะวิชยะทิ พุนาร ดฺะนัม
asau mayā hataḥ śatrur
 อเสา มยา หตห์ ศตฺรุรฺ
 อะเสา มะยา หะตะห์ ศัตรุร
 อโส มะยา ฮะทะฮ ชะทรุร
 haniṣye cāparān api
 หนิเษฺย จาปรานฺ อปิ
 หะนิษเย จาปะราน อะปิ
 ฮะนิชเย ชาพะราน อพิฺ
 īśvaro ’ham ahaṁ bhogī
 อีศฺวโร ’หมฺ อหํ โภคี
 อีศวะโร ’หัม อะหัง โภคี
 อีชวะโร ่ฮัม อฮัม โบฺกี
 siddho ’haṁ balavān sukhī
 สิทฺโธ ’หํ พลวานฺ สุขี
 สิทโธ ’หัง พะละวาน สุขี
 สิดโดฺ ่ฮัม บะละวาน สุคีฺฺ
āḍhyo ’bhijanavān asmi
 อาโฒฺย ’ภิชนวานฺ อสฺมิ
 อาฒโย ’ภิชะนะวาน อัสมิ
 อาดฺโย ่บิฺจะนะวาน อัสมิ
 ko ’nyo ’sti sadṛśo mayā
 โก ’โนฺย ’สฺติ สทฺฤโศ มยา
 โก ’นโย ’สติ สัทฤโศ มะยา
 โค ่นโย ่สทิ สะดริโช มะยา
 yakṣye dāsyāmi modiṣya
 ยกฺเษฺย ทาสฺยามิ โมทิษฺย
 ยักษเย ทาสยามิ โมทิษยะ
 ยัคชเย ดาสยามิ โมดิชยะ
 ity ajñāna-vimohitāḥ
 อิตฺยฺ อชฺญาน-วิโมหิตาห์
 อิตย อัชญานะ-วิโมหิตาห์
 อิทิ อกยานะ-วิโมฮิทาฮ

idam อิทมฺ อิทัม อิดัม — นี้, adya อทฺย อัทยะ อัดยะ — วันนี้, mayā มยา มะยา มะยา — โดยข้า, labdham ลพฺธมฺ ลัพธัม ลับดัฺม — กำไร, imam อิมมฺ อิมัม อิมัม — นี้, prāpsye ปฺราปฺเสฺย ปราปสเย พราพสเย — ข้าจะกำไร, manaḥ-ratham มนห์-รถมฺ มะนะห์-ระถัม มะนะฮ-ระทัฺม — ตามความปรารถนาของข้า, idam อิทมฺ อิทัม อิดัม — นี้, asti อสฺติ อัสติ อัสทิ — มี, idam อิทมฺ อิทัม อิดัม — นี้, api อปิ อะปิ อพิ — เช่นกัน, me เม เม เม — ของข้า, bhaviṣyati ภวิษฺยติ ภะวิษยะติ บฺะวิชยะทิ — จะเพิ่มพูนขึ้นในอนาคต, punaḥ ปุนห์ ปุนะห์ พุนะฮ — อีกครั้งหนึ่ง, dhanam ธนมฺ ธะนัม ดฺะนัม — ทรัพย์ สมบัติ, asau อเสา อะเสา อโส — นั้น, mayā มยา มะยา มะยา — โดยข้า, hataḥ หตห์ หะตะห์ ฮะทะฮ — ได้ถูกสังหาร, śatruḥ ศตฺรุห์ ศัตรุห์ ชะทรุฮ — ศัตรู, haniṣye หนิเษฺย หะนิษเย ฮะนิชเย — ข้าจะ สังหาร, ca จ จะ ชะ — เช่นกัน, aparān อปรานฺ อะปะราน อพะราน — บุคคลอื่น ๆ, api อปิ อะปิ อพิ — แน่นอน, īśvaraḥ อีศฺวรห์ อีศวะระห์ อีชวะระฮ — เจ้า, aham อหมฺ อะหัม อฮัม — ข้าเป็น, aham อหมฺ อะหัม อฮัม — ข้าคือ, bhogī โภคี โภคี โบฺกี — ผู้มีความสุขเกษมสำราญ, siddhaḥ สิทฺธห์ สิทธะห์ สิดดฺะฮ — สมบูรณ์, aham อหมฺ อะหัม อฮัม — ข้าเป็น, bala-vān พล-วานฺ พะละ-วาน บะละ — วาน-มีอำนาจ, sukhī สุขี สุขี สุคีฺ — ความสุข, āḍhyaḥ อาฒฺยห์ อาฒยะห์ อาดฺยะฮ — ร่ำรวย, abhijana-vān อภิชน-วานฺ อะภิชะนะ-วาน อบิฺจะนะ-วาน — รายล้อมไปด้วยญาติ ๆ ที่มีสกุล, asmi อสฺมิ อัสมิ อัสมิ — ข้าคือ, kaḥ กห์ กะห์ คะฮ — ผู้ซึ่ง, anyaḥ อนฺยห์ อันยะห์ อันยะฮ — คนอื่น, asti อสฺติ อัสติ อัสทิ — มี, sadṛśaḥ สทฺฤศห์ สัทฤศะห์ สะดริชะฮ — เหมือน, mayā มยา มะยา มะยา — ข้า, yakṣye ยกฺเษฺย ยักษเย ยัคชเย — ข้าจะทำพิธีบูชา, dāsyāmi ทาสฺยามิ ทาสยามิ ดาสยามิ — ข้าจะให้ทาน, modiṣye โมทิเษฺย โมทิษเย โมดิชเย — ข้าจะรื่นเริง, iti อิติ อิติ อิทิ — ดังนั้น, ajñāna อชฺญาน อัชญานะ อกยานะ — ด้วยอวิชชา, vimohitāḥ วิโมหิตาห์ วิโมหิตาห์ วิโมฮิทาฮ — ลุ่มหลง

คำแปล

คนมารคิดว่า ”วันนี้ข้ามีทรัพย์สมบัติมากแค่นี้ ข้าจะกอบโกยมากยิ่งขึ้นไปอีกตามแผนการ ปัจจุบันนี้เป็นของข้ามากเท่านี้และจะเพิ่มมากยิ่ง ๆ ขึ้นไปในอนาคตคนนี้เป็นศัตรูข้า ข้าได้สังหารมัน และศัตรูอื่น ๆ ก็จะถูกสังหารเช่นเดียวกัน ข้าเป็นเจ้าของทุกสิ่งทุกอย่าง ข้าเป็นผู้มีความสุขเกษมสำราญ ข้าสมบูรณ์ ข้ามีอำนาจ และมีความสุข ข้าเป็นคนที่รวยที่สุด รายล้อมไปด้วยญาติ ๆ ที่มีสกุลรุนชาติ ไม่มีผู้ใดมีอำนาจและมีความสุขเหมือนดังข้า ข้าจะทำพิธีบูชา ข้าจะทำบุญดังนั้นข้าจะรื่นเริง” เช่นนี้บุคคเหล่านี้ลุ่มหลงอยู่ในอวิชชา
