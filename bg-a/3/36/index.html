กรฺม-โยค

โศลก 36

arjuna uvāca
 อรฺชุน อุวาจ
 อัรชุนะ อุวาจะ
 อารจุนะ อุวาชะ
 atha kena prayukto ’yaṁ
 อถ เกน ปฺรยุกฺโต ’ยํ
 อะถะ เกนะ ประยุกโต ’ยัง
 อทฺะ เคนะ พระยุคโท ยัม
 pāpaṁ carati pūruṣaḥ
 ปาปํ จรติ ปูรุษห์
 ปาปัง จะระติ ปูรุษะห์
 พาพัมชะระทิ พูรุชะฮ
 anicchann api vārṣṇeya
 อนิจฺฉนฺนฺ อปิ วารฺษฺเณย
 อะนิจฉันน อะปิ วารษเณยะ
 อนิชชฺันน อพิ วารชเณยะ
 balād iva niyojitaḥ
 พลาทฺ อิว นิโยชิตห์
 พะลาท อิวะ นิโยชิตะห์
 บะลาด อิวะ นิโยจิทะฮ

arjunaḥ uvāca อรฺชุนห์ อุวาจ อัรชุนะห์ อุวาจะ อารจุนะฮ อุวาชะ — arjuna อรฺชุน อัรชุนะ อารจุนะ ตรัส, atha อถ อะถะ อทฺะ — จากนั้น, kena เกน เกนะ เคนะ — ด้วยอะไร, prayuktaḥ ปฺรยุกฺตห์ ประยุกตะห์ พระยุคทะฮ — กระตุ้น, ayam อยมฺ อะยัม อยัม — บุคคล, pāpam ปาปมฺ ปาปัม พาพัม — ความบาป, carati จรติ จะระติ ชะระทิ — ทำ,pūruṣaḥ ปูรุษห์ ปูรุษะห์ พูรุชะฮ — มนุษย์, anicchan อนิจฺฉนฺ อะนิจฉัน อนิชชฺัน — ไม่มีความ ปรารถนา, api อปิ อะปิ อพิ — ถึงแม้ว่า, vārṣṇeya วารฺษฺเณย วารษเณยะ วารชเณยะ — โอ้ ผู้สืบราชวงศ์ vṛṣṇi วฺฤษฺณิ วฤษณิ วริชณิ,balāt พลาตฺ พะลาต บะลาท — ด้วยกำลัง, iva อิว อิวะ อิวะ — ประหนึ่งว่า, niyojitaḥ นิโยชิตห์ นิโยชิตะห์ นิโยจิทะฮ — ปฏิบัติ

คำแปล

arjuna อรฺชุน อัรชุนะ อารจุนะ  ตรัสว่า  โอ้  ผู้สืบราชวงศ์  vṛṣṇi วฺฤษฺณิ วฤษณิ วริชณิ  อะไรคือสิ่งกระตุ้นให้คนทำบาปแม้จะไม่เต็มใจ  ประหนึ่งทำไปเพราะถูกบังคับ?

คำอธิบาย

สิ่งมีชีวิตซึ่งเป็นละอองอณูขององค์  bhagavān ภควานฺ ภะคะวาน บฺะกะวาน  เดิมทีเป็ทิพย์  บริสุทธิ์  และปราศจากมลทินทางวัตถุทั้งปวง  ฉะนั้น  โดยธรรมชาติจะไม่อยู่ภายใต้อำนาจแห่งความบาปในโลกวัตถุ  แต่เมื่อมาสัมผัสกับธรรมชาติวัตถุเราทำบาปนานัปการโดยไม่ลังเลใจ  แม้บางครั้งตนเองไม่ปรารถนา  ดังนั้น  arjuna อรฺชุน อัรชุนะ อารจุนะ  ทรงถาม  kṛṣṇa กฺฤษฺณ กฤษณะ คริชณะ  เช่นนี้จึงให้ความหวังมากเกี่ยวกับธรรมชาติอันวิปริตของสิ่งมีชีวิต  แม้สิ่งมีชีวิตบางครั้งไม่ต้องการทำบาปแต่ถูกบังคับให้ทำ  อย่างไรก็ดีอภิวิญญาณผู้ประทับอยู่ภายในหัวใจของเราทรงมิได้เป็นผู้กระตุ้นให้ทำบาปแต่เนื่องมาจากสาเหตุอื่น  ดังที่องค์  bhagavān ภควานฺ ภะคะวาน บฺะกะวาน  จะทรงอธิบายในโศลกต่อไป
